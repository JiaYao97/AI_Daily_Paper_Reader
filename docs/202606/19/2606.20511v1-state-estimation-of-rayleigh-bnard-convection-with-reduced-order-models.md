## 摘要
本文针对二维瑞利-贝纳德（RB）对流发展了一套状态估计框架，该框架将稳定的伽辽金降阶模型（ROM）与扩展卡尔曼滤波（EKF）相结合。ROM由线性化布西内斯克方程的可控性模态构建，为滤波预测步骤提供非线性动力学模型。直接数值模拟（DNS）用于生成数据同化的合成测量值。我们评估了滤波器在周期、准周期和混沌状态下的性能，表明滤波器能够高保真地跟踪最能量模态，并实现速度场时间平均重构误差低于14%、温度场低于9%。我们将基于ROM的EKF应用于一种混合模拟场景，其中系统状态从粗糙的类PIV速度测量值中同化。结果表明，仅速度观测就足以重构包括温度场在内的状态。最后，我们利用卡尔曼增益矩阵开发了一种贪心传感器布置策略，逐步去除信息量最少的传感器。该算法揭示了传感器类型之间的清晰层次结构，可用于推导骨架观测配置，并为哪些测量变量和空间位置对状态校正最具信息量提供指导。本框架具有通用性，可应用于其他二次伽辽金ROM进行状态估计。

## Abstract
In this work, we develop a state estimation framework for two-dimensional Rayleigh-Bénard (RB) convection that combines a stable Galerkin reduced-order model (ROM) with an extended Kalman filter (EKF). The ROM, constructed from controllability modes of the linearised Boussinesq equations, provides the nonlinear dynamical model for the filter prediction step. Direct numerical simulations (DNS) are used to generate synthetic measurements for data assimilation. We assess filter performance across periodic, quasiperiodic, and chaotic regimes, demonstrating that the filter tracks the most energetic modes with high fidelity and achieves time-averaged reconstruction errors below $14\%$ for velocity and $9\%$ for temperature. We apply the ROM-based EKF to a hybrid simulation scenario where the system state is assimilated from coarse PIV-like velocity measurements. It is shown that velocity observations alone suffice to reconstruct the state, including the temperature field. Finally, we exploit the Kalman gain matrix to develop a greedy sensor placement strategy that progressively removes the least informative sensors. The algorithm reveals a clear hierarchy among sensor types and can be used to derive skeletal observation configurations. It also provides guidance on which measurement variables and spatial locations are most informative for state correction. The present framework is general, and may be applied to other quadratic Galerkin ROMs for state estimation.

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将RB对流的Boussinesq方程投影到一组由线性化方程可控制性Gramian特征函数构成的**耦合正交模态**上，得到低维的伽辽金ROM（状态维度 $n \sim 10^2$）。该ROM能保持数值稳定，无需额外闭合模型。将ROM的常微分方程作为EKF的预测模型，利用其解析Jacobian进行线性化，再通过稀疏观测（速度、温度探针）在模态空间中执行标准EKF更新。
- **关键技术细节**：
  1. **模态基构造**：定义扩展状态 $\mathbf{X}=[u, v, \theta]^T$，分解为稳态温度剖面 $\theta_0(y)$ 与模态展开：$\mathbf{X}(t,\mathbf{x}) = [0,0,\theta_0(y)]^T + \sum_{j=1}^n c_j(t)\boldsymbol{\chi}_j(\mathbf{x})$。模态 $\boldsymbol{\chi}_j = [U_j, V_j, T_j]^T$ 由线性化方程的可控制性Gramian的特征函数给出，并满足加权的正交条件 $\langle \boldsymbol{\chi}_i, \boldsymbol{\chi}_j \rangle_c = \delta_{ij}$，其中权重 $\gamma^2 = 1.24$ 由DNS数据拟合确定。
  2. **ROM动态方程**：代入模态展开并取内积得到：
     $$ \frac{dc_i}{dt} = Pr\left(F_i^0 + \sum_j F_{ij}^1 c_j\right) + \frac{Pr}{\sqrt{Ra}}\sum_j D_{ij}^V c_j + \frac{1}{\sqrt{Ra}}\sum_j D_{ij}^T c_j - \sum_j L_{ij}c_j - \sum_{j,k} N_{ijk}c_j c_k $$
     其中各投影算子由内积定义。该方程可简写为 $\frac{d\mathbf{c}}{dt} = \mathbf{f}(\mathbf{c}; Ra, Pr)$。
  3. **EKF流程**：
     - **预测步**：$\hat{\mathbf{c}}_k^- = \Phi(\hat{\mathbf{c}}_{k-1})$，$P_k^- = F_k P_{k-1} F_k^T + Q$，其中 $F_k = I + J(\hat{\mathbf{c}}_{k-1})\Delta t$，$J$ 是ROM的解析Jacobian。
     - **更新步**：计算创新 $\mathbf{z}_k = \mathbf{y}_k - H\hat{\mathbf{c}}_k^-$，创新协方差 $S_k = H P_k^- H^T + R$，卡尔曼增益 $K_k = P_k^- H^T S_k^{-1}$，得到后验 $\hat{\mathbf{c}}_k = \hat{\mathbf{c}}_k^- + K_k \mathbf{z}_k$，后验协方差采用Joseph形式 $P_k = (I-K_k H)P_k^-(I-K_k H)^T + K_k R K_k^T$。
     - **观测矩阵** $H$ 由模态在探头位置的值构成，支持三种类型的传感器（水平速度、垂直速度、温度扰动）。

### 6. 论文的主要结论与发现

- 结合稳定ROM和EKF的框架能够以**低于14%的速度场误差和低于9%的温度场误差**在周期/准周期/混沌状态下高保真地重建RB对流状态。

- 贪心传感器布置策略揭示传感器重要性层级：**水平速度 > 垂直速度 > 温度**；此策略生成的稀疏配置（$m=12$）相比均匀网格在相同观测维度下误差显著降低（$Ec$ 从64%降至24%）。
- 数据估计的协方差矩阵 $\hat{Q},\hat{R}$ 相较于理想化的各向同性协方差，在所有三种状态下进一步降低了误差（例如混沌状态下 $Ec$ 从13.8%降至9.2%）。
- 噪声比率 $\beta=0.01$ 接近最优；增加观测密度（$m$）对速度重构改善更有效，而增加ROM维度（$n$）对温度重构改善更明显。

### 7. 优点

1. **ROM稳定性**：基于可控性模态的Galerkin ROM无需附加闭合模型即可保持数值稳定，避免了传统POD-Galerkin ROM常见的不稳定问题。
2. **计算效率**：EKF在低维模态空间运行（$n\sim10^2$），Jacobian解析已知，无需有限差分近似，运算成本远低于全阶EKF。
3. **传感器布局启发式**：基于卡尔曼增益的贪心移除策略无需额外仿真，利用已有信息就能识别重要测量变量和位置，且结果物理可解释（活跃波动区更重要）。
4. **通用性**：方法仅依赖二次Galerkin ROM的存在，可推广至其他流体（如圆柱绕流、混合层、Couette流等）。
5. **实验设计系统**：对噪声参数、模型/观测维度、动力学状态均进行了敏感性和消融分析，鲁棒性验证充分。

### 8. 不足与局限

1. **二维且中等瑞利数**：仅限于二维DNS，且 $Ra/Ra_c \leq 120$（对应 $Ra \approx 2\times10^5$），未推广至高Ra湍流或三维情形。

6. **仅适用于可构建二次Galerkin ROM的流动**：若流动不能有效降阶或ROM不稳定，则框架失效。
