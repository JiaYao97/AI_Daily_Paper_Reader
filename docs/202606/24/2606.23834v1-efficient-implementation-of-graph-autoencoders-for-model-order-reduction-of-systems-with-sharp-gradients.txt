# Effi cient implementation of graph autoencoders for model-order reduction of systems with sharp gradients 

Liam K. Magargal a, Parisa Khodabakhshi a

Abstract 

This study investigates the e ffi cient deployment of graph autoencoders, a class of graph neural networks (GNNs), for model-order reduction (MOR) of high-dimensional dynamical systems. The proposed framework leverages graph autoencoders to perform nonlinear dimensionality reduction, enabling low-dimensional representations of systems characterized by sharp gradients for which conventional linear approximations, such as proper orthogonal decompo-sition (POD), are inadequate. In prior work, the graph autoencoder was deployed in the geometric deep least-squares Petrov-Galerkin (GD-LSPG) framework to evolve the latent representation with time. Although GD-LSPG achieved substantial dimensionality reduction for systems with sharp gradients, it failed to achieve cost savings because the re-quired repeated evaluation of the Jacobian of the decoder introduced an operation count that scaled with the dimension of the original high-dimensional system. To address this limitation, this study introduces graph neural network latent space dynamics identification (GNN-LaSDI). Specifically, GNN-LaSDI employs an operator learning framework to directly approximate the temporal evolution of the graph autoencoder’s latent representation, thereby avoiding the costly repeated evaluation of the Jacobian of the decoder. The performance of GNN-LaSDI is assessed against both GD-LSPG and POD latent space dynamics identification (POD-LaSDI), which combines POD-based dimensional-ity reduction with operator learning. In addition to standard error metrics, this work presents a novel point cloud error metric specifically tailored to evaluate the accuracy of the identified locations of sharp gradients within the so-lution. The e ff ectiveness of the metric and the proposed MOR framework is demonstrated through two numerical experiments featuring sharp gradients: Kobayashi’s solidification model for two-dimensional (2D) nucleus growth and a Riemann solver for the 2D Euler equations modeling a bow shock generated by flow past a cylinder. For the Euler equations experiment, a constraint-enforcement strategy is incorporated into the graph autoencoder to ensure compliance with physical laws, such as positive pressure constraints. For the studied problems, GNN-LaSDI incurs a substantially lower computational cost than GD-LSPG, though it remains slightly more computationally expensive than POD-LaSDI. However, GNN-LaSDI achieves significantly greater accuracy than POD-LaSDI, thereby providing a balance between predictive accuracy and computational speedup. Additionally, the results indicate that the proposed point cloud error provides a more intuitive and informative measure of reduced-order model accuracy in regions with sharp gradients than conventional error metrics. 

Keywords: Model-order reduction, Latent space dynamics identification, Graph autoencoders, Operator learning, Point cloud error, Sharp gradients 

1. Introduction 

Outer-loop applications in computational mechanics, such as optimization, uncertainty quantification, and control, require repeated evaluations of a numerical model. When the underlying numerical model is large-scale, such as those obtained from the semidiscretization of partial di ff erential equations (PDEs) using finite di ff erence, finite volume, or finite element methods, the associated computational cost can become prohibitively expensive. Model-order reduc-tion (MOR) addresses this challenge by constructing e ffi cient low-dimensional approximations of the original high-dimensional system. MOR techniques generally involve two stages. In the o ffl ine stage, high-dimensional solution

data generated by the full-order model (FOM) are collected and used to construct a low-dimensional representation of the system dynamics. This reduced representation is then employed to derive a reduced-order model (ROM) that approximates the behavior of the original system. In the subsequent online stage, the ROM is deployed to e ffi ciently evaluate new queries in outer-loop applications. Traditionally, low-dimensional representations of the high-dimensional state solutions are constructed using linear manifolds, most notably through proper orthogonal decomposition (POD) [1]. While linear manifolds have demon-strated remarkable success across a wide range of engineering applications [2–4], it is well known that MOR ap-proaches based on linear manifolds often struggle to achieve practical dimensionality reduction for systems charac-terized by a slow-decaying Kolmogorov n-width [5–7], including solutions containing sharp gradients, shocks, and moving interfaces. To address this limitation of linear manifolds, several studies have sought to employ nonlinear manifolds, such as quadratic manifolds [8, 9], closure models [10, 11], nonlinear probabilistic manifold decomposi-tions [12], and autoencoder-based methods [13–19]. Such studies demonstrate that nonlinear manifold approxima-tions of high-dimensional state solutions with a slow-decaying Kolmogorov n-width can more significantly reduce the dimensionality of the system than linear manifold approximations. To evaluate the time evolution of the low-dimensional representations, many MOR approaches project the high-dimensional system onto a reduced basis using Galerkin, Petrov-Galerkin [20], least-squares Petrov-Galerkin (LSPG) [13, 21–23], and adjoint Petrov-Galerkin [24] projections. For systems with predominantly linear dynamics, dimen-sional compression alone is often su ffi cient to achieve substantial computational savings [25]. In contrast, for systems governed by nonlinear dynamics, additional treatment is typically required to achieve cost savings, because evaluating the projected nonlinear terms incurs an operation count complexity that scales with the dimension of the FOM. To address this issue, hyper-reduction techniques have been developed to approximate the nonlinear terms using sparse sampling strategies. Representative examples include the empirical interpolation method [26], the discrete empirical interpolation method [27, 28], Gauss-Newton with approximated tensors [29], and the energy-conserving sampling and weighting method [30]. Rather than approximating the projected nonlinear terms, some studies aim to leverage operator learning strategies such as dynamic mode decomposition [31] and operator inference (OpInf) [32]. OpInf, which is particularly well-suited for systems with polynomial nonlinearities, leverages the fact that the polynomial structure is preserved under linear projection [33] to infer low-dimensional operators directly from FOM solutions. For systems that do not naturally admit a polynomial nonlinearity, lifting transformations can be introduced to refor-mulate the dynamics in a polynomial form amenable to OpInf [4, 34–38]. More recently, operator learning strategies have been extended to settings using both linear and nonlinear manifolds. For example, latent space dynamics iden-tification (LaSDI) constructs a ROM using a general library of functions by learning latent state dynamics from the training data recovered from the FOM projected onto the latent space [39, 40]. While the deployment of linear mani-folds to OpInf has been shown to preserve the polynomial structure of the dynamical system [33, 41], LaSDI extends the low-dimensional dynamics to contain any form of function. In the authors’ previous study [15], a graph autoencoder tailored to perform nonlinear dimensionality reduc-tion upon systems modeled by unstructured meshes was developed. Specifically, because unstructured meshes can be naturally modeled as graphs, the developed graph autoencoder natively accounted for systems modeled by un-structured meshes, thereby providing improved flexibility over traditional convolutional neural network (CNN) based autoencoders, which are inherently limited to systems employing structured meshes. The graph autoencoder was implemented into a nonlinear manifold LSPG framework called geometric deep least-squares Petrov-Galerkin (GD-LSPG). Although GD-LSPG achieved substantial dimensionality reduction for systems exhibiting sharp gradients, the approach remained subject to two key limitations. First, GD-LSPG failed to deliver computational savings because the required evaluation of the Jacobian of the decoder at each Newton iteration scaled with the FOM dimension, which proved costly over multiple iterations. Second, conventional error metrics, similar to those employed in [9, 13, 38], were used to assess the accuracy of GD-LSPG. However, it was shown that these metrics alone are often insu ffi cient for evaluating ROM performance on systems exhibiting sharp gradients, as they do not directly quantify the accuracy of localized features such as interfaces and shocks. The present work seeks to address both of these limitations. Toward overcoming the first limitation, this study introduces a regularized LaSDI scheme, referred to as graph neural network latent space dynamics identification (GNN-LaSDI), to directly approximate the temporal evolution of the graph autoencoder latent variables. By learning the latent space dynamics directly, the proposed approach bypasses the need for repeated evaluation of the Jacobian of the decoder and thereby enables significant computational savings. To address the second limitation, this study introduces a point cloud error metric that directly quantifies the 2errors in the predicted location of the sharp gradients present in the solution. Through two numerical experiments with sharp gradient features, this metric is demonstrated to provide a more intuitive assessment of ROM accuracy than conventional error metrics. In the first experiment, the studied MOR frameworks are applied to Kobayashi’s solidification model for two-dimensional (2D) homogeneous phase evolution to assess their ability to capture the evolution of the solid-liquid interface. In the second numerical experiment, the methods are applied to a Riemann solver for the 2D Euler equations modeling the bow shock generated by flow past a cylinder, with emphasis placed on accurately predicting the shock location. For the latter problem, a strategy for ensuring the physical admissibility of the state solutions in the graph autoencoder-based ROM predictions is presented. The primary contributions of this work are summarized as follows: 1. An operator learning framework based on LaSDI is developed to approximate the latent space dynamics of graph autoencoders, enabling a computationally e ffi cient MOR method. 2. A novel point cloud error metric is presented to directly quantify the accuracy of the predicted sharp gradient locations, such as interfaces and shocks, in ROM solutions. 3. In the second numerical example, a physics-informed strategy is proposed to enforce physical admissibility (e.g., preservation of positive pressure) in graph autoencoder predictions. The remainder of the paper is organized as follows. Section 2 introduces the form of the FOM considered in this study and provides background on the graph autoencoder. Section 3 formulates the regularized operator learning procedure used to construct the LaSDI ROMs. Section 4 discusses the error metrics employed in this study, including the novel point cloud error specifically suited to better capture discrepancies near sharp gradients. Section 5 presents two numerical experiments used to evaluate both the accuracy of the presented ROM framework and the e ff ectiveness of the point cloud error for sharp gradients. Finally, Section 6 summarizes the main conclusions of this work and outlines directions for future work.

2.2. Nonlinear dimension reduction via graph autoencoders 

A wide range of nonlinear mappings has been proposed in recent years to construct low-dimensional latent spaces for MOR on nonlinear systems. Existing approaches include quadratic manifolds [8, 9], closure models [10, 11], and autoencoder-based techniques [13, 16–18]. In this study, graph autoencoders are employed to perform nonlinear dimensionality reduction and extract low-dimensional latent representations from high-dimensional state solution 3data. A detailed description of the graph autoencoder architecture used in this study is provided in [15]; here, only key concepts relevant to the present framework are briefly reviewed. Graph autoencoders are a class of deep learning architecture designed to learn low-dimensional representations of graph-structured data. A graph is defined as the tuple G = {V, E}, where V denotes the set of nodes, and E denotes the set of edges that encode relationships between nodes. A node feature matrix, X ∈ R|V|× NF , can be used to represent information about each node in the graph, where the ith row of X corresponds to the feature vector associated with the 

ith node in V, |V| represents the number of nodes in node set V, and NF ∈ N denotes the number of features assigned to each node. In the context of large-scale dynamical systems arising in engineering applications, graphs provide a natural rep-resentation of both structured and unstructured meshes by modeling collocation points as nodes in the graph and pre-scribing edges to connect neighboring collocation points according to the mesh topology. Representing the mesh as a graph enables the direct application of graph autoencoders for nonlinear dimensionality reduction of high-dimensional state solution data, yielding low-dimensional latent representations. A graph autoencoder consists of two main com-ponents: an encoder and a decoder (see Figure 1). The encoder, Enc : x 7 → ˆx with Enc : RN → Rn, where n ≪ N,compresses the high-dimensional input state x ∈ RN into a low-dimensional latent representation ˆ x ∈ Rn. On the other hand, the decoder, Dec : ˆ x 7 → ˜x with Dec : Rn → RN , reconstructs an approximation ˜ x ∈ RN of the original high-dimensional state from the latent representation ˆ x ∈ Rn. More specifically, the encoder and decoder each consist of a sequence of layers. The graph autoencoder achieves compression using a hierarchy of nℓ graphs, G0, . . . , Gnℓ −1, where G0 denotes the original graph with a node set that represents the collocation points of the physical domain. Each successive graph 4in the hierarchy contains fewer nodes than its predecessors, i.e., |Vi+1| < |Vi|, where |Vi| denotes the number of nodes in Gi. This hierarchical reduction in graph size enables progressive compression of the state representation. The functions in the encoder use this hierarchy to reduce the dimension of the state solution to its low-dimensional latent representation, while the functions in the decoder use the hierarchy of graphs in reverse order to reconstruct the high-dimensional state solution from its low-dimensional representation. In other words, the ith graph in the encoder serves as the ( nℓ − i − 1) th graph in the decoder. Figure 1 demonstrates the use of the hierarchy of graphs within the graph autoencoder architecture. Several methods can be used to construct the hierarchy of graphs [42–45]. This study uses a hierarchical spectral clustering algorithm, also used in [15]. At each level of the hierarchy, the graph autoencoder performs a message passing (MP) operation to perform filtering (i.e., extracting useful information), where the MP operation contains trainable weights and biases. Numerous choices for MP operations exist in the literature, but in this study, the mean-aggregation MP layer found in [46] is deployed. The final layer of the encoder and the first layer of the decoder consist of a fully-connected (FC) layer to encode the node feature matrix associated with the graph Gnℓ −1 to the latent state vector, ˆ x, and vice versa. Further details regarding each layer in the graph autoencoder are provided in Appendix A. In the authors’ previous work [15], GD-LSPG was developed to approximate the temporal evolution of the low-dimensional latent state vector, ˆ x. Specifically, the method employed a nonlinear manifold LSPG projection con-structed using the graph autoencoder. A brief overview of GD-LSPG is provided in Appendix A, however, the key limitation motivating the present work is summarized below. Although GD-LSPG was shown to achieve significant dimensionality reduction, the repeated evaluation of the Jacobian of the decoder impeded cost savings. In particular, the operation count associated with evaluating the Jacobian of the decoder scaled with the dimension of the FOM. In conventional projection-based ROMs employing linear trial bases, this is typically addressed through hyper-reduction techniques, which sparsely sample terms in the high-dimensional system. Hyper-reduction can also be applied to autoencoder-based ROMs when shallow decoder architectures are employed [47]. However, the graph autoencoder presented in this study uses a deep decoder architecture, rendering e ffi cient hyper-reduction more challenging. As discussed in [30], in the absence of hyper-reduction, ROMs employing LSPG schemes achieve computational cost savings primarily in two scenarios. In the first scenario, savings may be achieved if the cost of the FOM is dominated by the cost of inverting the Jacobian of the time-discrete high-dimensional residual, while the cost to set up and solve the low-dimensional system at each Newton iteration is considerably lower. In the second scenario, savings may be obtained when the LSPG ROM remains stable and accurate using larger time step sizes than the FOM, thereby reducing the total number of time steps required for simulation. If neither condition is satisfied, GD-LSPG may not achieve cost savings. Motivated by this limitation, the remainder of this manuscript focuses on alternative strategies for approximating the temporal evolution of the low-dimensional latent representation of the graph autoencoder.

4. Evaluation of model prediction accuracy and model e ffi ciency 

Evaluating the accuracy of ROM predictions for systems exhibiting sharp gradients, such as interfaces and shocks, has recently emerged as an active area of research [49]. Conventional error metrics used in the MOR literature [13, 32, 50] are often not well-representative of a ROM’s predictive accuracy in such settings, since even small discrepancies 6in the location of a sharp gradient can produce large global errors despite the solution remaining qualitatively accurate [15, 49]. Furthermore, these metrics often fail to adequately capture the severity of errors associated with di ff used or poorly resolved sharp gradients, making it di ffi cult to distinguish between solutions that accurately preserve sharp features and those that do not. To address these limitations, this study introduces a point cloud error metric particularly suited for quantifying the accuracy of the predicted sharp gradient location. Section 4.1 summarizes the conventional error metrics employed in this study, while Section 4.2 presents the proposed point cloud error metric. In addition to predictive accuracy, the computational e ffi ciency of each ROM is assessed using the speedup factor speedup factor = wall-clock time for FOM wall-clock time for ROM , (13) which compares the online wall-clock time of the ROM against that of the FOM. Larger speedup factors indicate higher computational e ffi ciency.

Two numerical experiments are considered in this study. The first example examines MOR for Kobayashi’s solidification model for 2D isotropic nucleus growth [56], where the feature of interest is the moving interface between the solid and liquid phases. 6. Conclusions and discussion 

This study introduced GNN-LaSDI, a MOR framework that combines graph autoencoder dimensionality reduc-tion with latent space operator learning. The proposed approach was motivated by a key limitation of the previously developed GD-LSPG framework [15]: although graph autoencoders attain significant dimensionality reduction capa-bilities for transport-dominated problems with sharp gradients, the repeated evaluation of the Jacobian of the decoder required by nonlinear manifold LSPG projections can significantly limit online computational e ffi ciency. By replac-ing the nonlinear LSPG projection with LaSDI, GNN-LaSDI eliminates this computational burden while retaining the state compression capabilities of the graph autoencoder. In addition, this work presented a point cloud error metric designed to quantify errors in the predicted locations of sharp gradient features. Unlike conventional state-based error metrics, which can be dominated by small spatial shifts in interfaces and shocks, the proposed metric evaluates the spatial accuracy of the extracted feature location through a point cloud representation. Consequently, it provides a complementary measure of ROM fidelity that is well suited for transport-dominated systems. The proposed framework was assessed on two nonlinear systems characterized by evolving sharp gradients: Kobayashi’s phase field model for 2D solidification and the 2D Euler equations modeling a bow shock generated by flow past a cylinder. Across both numerical experiments, GNN-LaSDI and GD-LSPG consistently outperformed POD-LaSDI in preserving the evolution of interfaces and shocks. In the solidification problem, GNN-LaSDI achieved state prediction and point cloud errors comparable to those obtained with GD-LSPG while providing speedup fac-tors approximately two orders of magnitude larger. Similar trends were observed for the bow shock problem, where GNN-LaSDI accurately captured shock evolution over the parameter domain and maintained point cloud errors com-parable to GD-LSPG. The proposed constraint incorporated into the graph autoencoder architecture improved the GD-LSPG framework by ensuring physically admissible reconstructions for the second numerical example. A con-sistent observation across both numerical examples is that POD-LaSDI achieved the largest speedup factors, but at the expense of significantly larger state prediction and point cloud errors. Conversely, GD-LSPG generally yielded 23 the highest computational cost. GNN-LaSDI, on the other hand, demonstrated a balance between predictive accuracy and computational performance. Although the proposed point cloud error metric proved e ff ective for evaluating the accuracy of interface and shock locations, it does not assess the accuracy of the state values on either side of the identified shock feature. Consequently, it should be viewed as a complementary metric rather than a replacement for conventional state-based error metrics. Future work will investigate developing feature-aware error metrics that simultaneously account for both feature accuracy and state reconstruction fidelity. Another avenue for future work is to investigate a richer library of functions utilized by LaSDI. Many dynamical systems admit parsimonious representations in appropriately chosen coordinates. Inspired by SINDy [48], future work may investigate combining Tikhonov and Lasso regularization [59] within the LaSDI framework to facilitate the selection of representative terms from a richer library of candidate functions.

Appendix A. Fundamentals of Geometric deep least-squares Petrov-Galerkin 

The deep LSPG framework [13] employed a nonlinear manifold LSPG projection using a CNN-based autoen-coder. While CNN-based autoencoders demonstrated exceptional dimensionality reduction capabilities for systems exhibiting sharp gradients [13, 17, 60], they were not readily extendable to unstructured meshes. To overcome this limitation, the authors’ previous work [15] introduced a graph autoencoder (also used in this study), enabling nonlin-ear manifold MOR on unstructured meshes through GNN architectures. A brief overview of GD-LSPG is presented here. Specifically, Appendix A.1 summarizes the encoder and decoder architectures presented in Section 2.2, while Appendix A.2 provides a summary of the nonlinear manifold LSPG projection used to compute the temporal evolution of the latent variables.

All graph autoencoder models are trained using PyTorch [61] and PyTorch-Geometric [62] on an NVIDIA L40S GPU. The weights and biases of all models are initialized with Xavier initialization [63]. For both experiments, the loss function defined in (4) is minimized using the Adam optimizer [64] with a learning rate of 10 −4. The MP operation is implemented using the SAGEConv layer [46]. Exponential linear unit (ELU) [65] activation functions are applied after the FC layer in the decoder and after each MP operation in both the encoder and the decoder, with the exception of the final MP operation in the decoder. In the experiment using Kobayashi’s solidification model, no activation function is applied after the final MP operation in the decoder. Alternatively, the modified postprocessing layer deployed in the 2D Euler equations experiment applies a sigmoid activation function after the final MP operation in the decoder. The FOM implementation for Kobayashi’s solidification model for 2D homogeneous phase evolution is executed on four Intel ® Xeon ® Platinum 8358 CPU @ 2.60GHz ICE LAKE CPU cores using NumPy [66]. Since backward Euler time integration is employed, a Newton solver is used to minimize the high-dimensional residual. Sparse linear algebra routines from SciPy [67] are used to e ffi ciently solve the resulting linear system repeatedly. The FOM implementation for the Riemann solver for the 2D Euler equations is likewise executed on four Intel ® Xeon ® Platinum 8358 CPU @ 2.60GHz ICE LAKE CPU cores using NumPy [66] together with Numba’s just-in-time compiler [68]. For the Kobayashi’s solidification problem, the graph autoencoder is trained using a training /validation split con-sisting of 12500 snapshots and 25 validation snapshots obtained form the FOM. For the bow shock problem governed by the 2D Euler equations, the graph autoencoder is trained using 5000 training snapshots, and 511 validation snap-shots. The snapshots are represented using the conserved variables ( ρ, ρ u, ρ v, and ρE). In both experiments, the network is trained for 2500 epochs using a batch size of 20. To construct the POD basis for the POD-LaSDI ROM, a thin singular value decomposition (SVD) is performed on the snapshot matrix containing all FOM snapshots (i.e., 12525 for the first experiment, and 5511 for the second experiment). The hyperparameters used for the GD-LSPG simulations are summarized next. For Kobayashi’s solidification model, the Newton update in (A.9) uses a constant step length η j = 1.0 for all updates. For the bow shock problem, the Newton update (A.9) is computed using a step length of η j = 1.0, however, if convergence is not achieved, the step length is reduced by 90% for every 10 iterations. Both experiments use the covergence tolerance κ = 10 −3 in (A.10). Each ROM is executed using hardware chosen to optimize online computational performance. For GNN-LaSDI and POD-LaSDI, the temporal evolution of the latent space representation is computed using NumPy [66] on four Intel ® Xeon ® Platinum 8358 CPU @ 2.60GHz ICE LAKE CPU cores.
