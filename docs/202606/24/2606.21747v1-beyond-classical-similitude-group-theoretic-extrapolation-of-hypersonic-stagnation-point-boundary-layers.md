## 摘要
受将地面实验结果外推至高速飞行条件的需要驱动，我们提出了高超声速驻点边界层的李等价对称性分析。我们展示了等价对称性在高超声速中具有物理相关性的一组耦合常微分方程上的应用。通过允许物性定律随自变量和因变量变换，在相似性约化的驻点常微分方程公式中导出的不变量，识别出可用于将实验室尺度预测外推至飞行的非线性映射族。在实践中，实施这些映射需要实验室尺度的常微分方程解以及实验室和飞行侧的热化学物性数据，这些数据通常可从现有数据库获得。这些映射是针对边界层内相似性约化的无量纲温度导出的，并且在一系列相关案例中显示出与独立计算的飞行解一致。

## Abstract
Motivated by the need to extrapolate the results from ground-based experiments to the conditions of high-speed flight, we present the Lie equivalence symmetry analysis of the hypersonic stagnation-point boundary layers. We demonstrate the application of the equivalence symmetry on the set of coupled ODEs which are physically relevant in hypersonics. By allowing the property laws to transform along with the independent and dependent variables, the invariants derived within the similarity-reduced stagnation-point ODE formulation, identify families of non-linear maps that can be used to extrapolate the laboratory-scale predictions to flight. In practice, implementing these maps requires the laboratory-scale ODE solution together with the lab- and flight-side thermochemical property data, which are generally available from existing databases. The maps are derived for the similarity-reduced non-dimensional temperature across the boundary layer and are shown to {collapse with independently computed flight solutions} for a range of relevant cases.

## 论文提出的方法论

### 核心思想
- 使用**李等价对称性**（Lie equivalence symmetry）处理高超声速驻点边界层的相似约化常微分方程组。
- 与经典李对称不同，等价对称不仅缩放自变量和因变量（$\eta, F, V, g$），还允许**物性律（$R_\rho, C_p, \ell_0, \chi$）**随温度 $g$ 变换，从而将整个问题类映射到另一个。
- 关键步骤是求解包含物性律的线性化对称条件，导出无穷小生成元，进而获得有限变换。

### 关键技术细节
- **方程组**：从质量、动量、能量守恒出发，经相似变换得到关于归一化速度 $F$、流函数变量 $V$、温度 $g$ 的常微分方程组 (2.2)。
- **物性律**：$\ell_0 = \rho\mu/(\rho_e\mu_e)$，$\chi = \lambda\rho/(\rho_e\mu_e)$，$R_\rho = \rho_e/\rho$，$C_p$ 为比热，均依赖于 $g$（和边缘温度 $T_e$）。
- **对称推导**：
  - 扩展相空间，引入物性律作为额外因变量，并施加辅助方程 $\partial_\eta a_i = \partial_F a_i = \partial_V a_i = 0$。
  - 将二阶延长无穷小生成元投影到各方程，分离独立导数项，得到确定方程组。
  - 解出无穷小为：
    - $\phi_\eta = C_0 \eta$，$\phi_V = A_0 V$，$\phi_F = (A_0 - C_0)F$，$\phi_g = \phi_g(g)$
    - $\phi_{R_\rho} = 2(A_0 - C_0)R_\rho$，$\phi_{\ell_0} = (A_0 + C_0)\ell_0$
    - $\phi_\chi = [K_0 - d\phi_g/dg]\chi$，$\phi_{C_p} = [K_0 - A_0 - C_0 - d\phi_g/dg]C_p$
- **有限映射与不变量**：
  - 定义雅可比 $J = \partial\hat{g}/\partial g$，积分得到关键不变量：
    $$ \hat{\chi}(\hat{g}) = \chi(g) e^{K_0\varepsilon} J^{-1} $$
  - 沿边界层积分得到：
    $$ \int_{\hat{g}_{\text{内}}}^1 \hat{\chi}(s)\,ds = e^{K_0\varepsilon} \int_{g_{\text{内}}}^1 \chi(g)\,dg $$
  - 定义 $H_F(\hat{g}) = \int_{\hat{g}}^1 \chi_F(s)\,ds$，$H_L(g) = \int_g^1 \chi_L(s)\,ds$，则
    $$ \hat{g} = H_F^{-1}\big( e^{K_0} H_L(g) \big) $$
  - 常数 $K_0$ 由壁面条件确定：$K_0 = \ln\big( H_F(g_w^F) / H_L(g_w^L) \big)$。

### 算法流程
1. 使用 PlasFlow-Solver 计算实验室侧驻点边界层温度剖面 $g_L(\eta)$。
2. 从热化学数据库（如 Mutation++）获取实验室和飞行侧 $\chi(g)$（导热系数×密度/($\rho_e\mu_e$)）。
3. 计算积分 $H_L(g)$ 和 $H_F(g)$（分别用实验室和飞行物性数据）。
4. 由壁面条件确定常数 $K_0$。
5. 对每个 $g_L$ 点，计算目标值 $H_\text{target}=e^{K_0} H_L(g_L)$，通过插值求解 $\hat{g} = H_F^{-1}(H_\text{target})$。
6. 得到映射后的温度剖面 $\hat{g}(\hat{\eta})$（$\hat{\eta}=e^{C_0}\eta$，本文取 $C_0=0$）。
