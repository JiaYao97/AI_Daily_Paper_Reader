## 摘要
本文针对二维瑞利-贝纳德（RB）对流，开发了一种结合稳定伽辽金降阶模型（ROM）和扩展卡尔曼滤波（EKF）的状态估计框架。该ROM由线性化布西内斯克方程的可控性模态构造，为滤波预测步骤提供非线性动力学模型。直接数值模拟（DNS）用于生成数据同化的合成观测值。我们评估了周期、准周期和混沌状态下的滤波性能，表明滤波能以高保真度追踪最能量模态，并实现速度和温度的时间平均重构误差分别低于14%和9%。我们将基于ROM的EKF应用于混合模拟场景，其中系统状态由类似PIV的稀疏速度观测值同化。结果表明，仅凭速度观测就足以重构状态，包括温度场。最后，我们利用卡尔曼增益矩阵开发了一种贪婪传感器布置策略，逐步移除信息量最少的传感器。该算法揭示了传感器类型之间的清晰层次结构，可用于推导骨架观测配置。它还为状态校正中最具信息量的测量变量和空间位置提供了指导。本文框架具有通用性，可应用于其他二次伽辽金ROM进行状态估计。

## Abstract
In this work, we develop a state estimation framework for two-dimensional Rayleigh-Bénard (RB) convection that combines a stable Galerkin reduced-order model (ROM) with an extended Kalman filter (EKF). The ROM, constructed from controllability modes of the linearised Boussinesq equations, provides the nonlinear dynamical model for the filter prediction step. Direct numerical simulations (DNS) are used to generate synthetic measurements for data assimilation. We assess filter performance across periodic, quasiperiodic, and chaotic regimes, demonstrating that the filter tracks the most energetic modes with high fidelity and achieves time-averaged reconstruction errors below $14\%$ for velocity and $9\%$ for temperature. We apply the ROM-based EKF to a hybrid simulation scenario where the system state is assimilated from coarse PIV-like velocity measurements. It is shown that velocity observations alone suffice to reconstruct the state, including the temperature field. Finally, we exploit the Kalman gain matrix to develop a greedy sensor placement strategy that progressively removes the least informative sensors. The algorithm reveals a clear hierarchy among sensor types and can be used to derive skeletal observation configurations. It also provides guidance on which measurement variables and spatial locations are most informative for state correction. The present framework is general, and may be applied to other quadratic Galerkin ROMs for state estimation.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
- 使用线性化Boussinesq方程的可控性Gramian的特征函数构造正交耦合模态基（包含速度和温度成分），通过Galerkin投影得到非线性ROM，该ROM在截断后依然保持数值稳定，不需要闭合格子。

### 关键技术细节
- **模态基构造**：将状态向量 $X = [u, v, \theta]^T$ 分解为稳态平均温度 $\theta_0(y)=1-y$ 和扰动部分，扰动用正交模态 $\chi_j(x)=[U_j, V_j, T_j]^T$ 展开。引入加权内积 $\langle X_i, X_j \rangle_c$ 平衡动能和热能的贡献，权重 $\gamma^2=1.24$ 由DNS数据最小二乘拟合确定。模态通过求解每个水平波数下的Sylvester方程获得可控性Gramian的特征函数。
- **ROM形式**：将模态分解代入控制方程并投影，得到模态振幅 $c_j$ 的常微分方程：$$\frac{dc_i}{dt} = Pr(F0_i + \sum_j F1_{ij}c_j) + \frac{Pr}{\sqrt{Ra}}\sum_j DV_{ij}c_j + \frac{1}{\sqrt{Ra}}\sum_j DT_{ij}c_j - \sum_j L_{ij}c_j - \sum_{j,k} N_{ijk}c_jc_k.$$ 该方程写成紧凑形式 $\dot{c}=f(c;Ra,Pr)$。二次非线性允许解析计算雅可比矩阵 $J$，用于EKF的线性化。
- **扩展卡尔曼滤波（EKF）**：
  - **预测步骤**：将ROM从 $t_{k-1}$ 积分到 $t_k$ 得到预测状态 $\hat{c}^-_k$；预测协方差 $P^-_k = F_k P_{k-1} F_k^T + Q$，其中 $F_k = I + J(\hat{c}_{k-1})\Delta t$，$Q$ 为过程噪声协方差。
  - **更新步骤**：当观测 $y_k$ 到达时，计算创新 $z_k = y_k - H\hat{c}^-_k$，创新协方差 $S_k = HP^-_k H^T + R$，卡尔曼增益 $K_k = P^-_k H^T S^{-1}_k$。后验状态 $\hat{c}_k = \hat{c}^-_k + K_k z_k$，后验协方差用Joseph形式更新。
  - **观测矩阵 $H$**：由模态在探头位置的值构成，包含速度分量和温度扰动分量的子矩阵。
- **贪心传感器移除算法**：
  1. 在当前传感器配置下运行EKF，记录每个时刻的卡尔曼增益 $K_k$。
  2. 计算每个测量通道的长期重要性得分 $S_j = \sqrt{\sum_{k}\sum_i K_{k,ij}^2}$。
  3. 移除得分最低的传感器，更新 $H$ 和 $R$，重复直到观测维度达到目标或误差超限。

## 6. 论文的主要结论与发现

- **EKF+稳定ROM可有效估计RB对流状态**：在周期（R=40）、准周期（R=80）、混沌（R=120）三种 regime 下，使用粗网格（$m=48$）均能跟踪主导模态，时间平均重构误差：速度 $<14\%$，温度 $<9\%$。滤波误差 $E_c$ 随瑞利数增加而轻微上升（8.9%→13.8%）。
- **仅用速度观测可重构温度场**：在混沌 regime，使用 $16\times16$ 速度探头（$m=512$），重构误差均在 $10\%$ 以内（温度误差 $e_\theta \approx 9\%$）。表明ROM耦合了速度-温度动力学，无需直接温度测量。
- **贪心传感器移除揭示传感器重要性层次**：基于时间平均卡尔曼增益范数的贪心算法优先移除温度传感器，其次垂直速度，最后保留水平速度传感器。保留的传感器倾向于聚集在壁面附近高速度涨落区域。该算法明显优于同维数的均匀粗网格：在 $m=12$ 时，贪心策略的滤波误差 $E_c=24.0\%$，远低于粗网格的 $64.0\%$。
- **数据估计的噪声协方差进一步降低误差**：相比于各向同性人工设定，使用从DNS残差估计的 $Q$ 和 $R$ 可使所有 regime 下的误差下降（混沌 case 中 $E_c$ 从13.8%降至9.2%）。这些协方差矩阵的非对角元素显示误差并非各向同性或互不相关。
- **敏感性分析总结**：模型-测量噪声比 $\beta=0.01$ 接近最优；提高观测密度比提高ROM维度更能改善速度重构，而温度重构更依赖ROM的保真度。

## 7. 优点：方法或实验设计上有哪些亮点

- **稳定ROM的新颖性**：使用可控性模态而非POD快照，ROM截断后自发稳定，无需闭合格子，这是实现可靠状态估计的关键。
- **联合状态估计框架的通用性**：可应用于任何具有二次Galerkin ROM的系统，不限于RB对流。
- **仅速度测量重构温度场**：展示了ROM作为物理约束的内插能力，对实验数据同化（如PIV结合模拟）有实际价值，可减少温度探头需求。
- **贪心传感器优化策略**：基于EKF运行中已有的卡尔曼增益矩阵，无额外计算代价，且揭示物理含义（水平速度最重要）。方法简洁实用，能显著提升有限传感器数量下的估计性能。
- **跨动力学 regime 评估**：系统测试了周期、准周期、混沌三种状态，证明方法鲁棒性，这是许多先前工作所缺乏的。
- **噪声协方差的数据驱动估计**：验证了模型误差和测量误差的非各向同性特征，并给出了更优的协方差，提升精度。

## 8. 不足与局限：包括实验覆盖、偏差风险、应用限制等

- **缺乏与其他方法的直接对比**：未与POD-ROM+EKF、EnKF、nudging等基准方法在相同设置下定量比较，无法断言性能优势。
- **仅使用合成数据**：所有观测来源于DNS的投影/抽取，未考虑真实实验噪声（如PIV散斑噪声、温度探头热噪声）、时间相关误差、空间缺失等，实用性验证不足。
- **二维低瑞利数限制**：$Ra \approx 2\times10^5$ 且 $Pr=10$，远低于真正湍流对流（$Ra>10^9$）。扩展到高瑞利数或三维时，ROM维度需求剧增，EKF的协方差传播可能失效。
- **固定普朗特数和水平长宽比**：$Pr=10$ 对应中等普朗特数，$L_x=2$ 仅为两倍单元宽度，更宽域或不同液体（如液态金属 $Pr\ll1$）可能改变结论。
- **EKF对非高斯噪声的假设**：假定过程和测量噪声为零均值高斯白噪声，但实际截断误差可能具有非高斯、有色特征，论文未处理时间相关性（仅指出为未来工作）。
- **贪心算法的次优性**：贪心依次移除，未考虑传感器组合的协同效应，可能陷入局部最优。未与更严密的优化方法（如凸松弛、互相信息最大化）对比。
- **缺乏实验数据集和代码可复现性**：代码开源（GitHub）提升可复现性，但DNS数据和完整EKF参数设定（如时间步长、自适应容差）可能需额外努力才能完全复现。
- **资源算力未报告**：无法评估方法在实际应用中的计算成本与可行性。
