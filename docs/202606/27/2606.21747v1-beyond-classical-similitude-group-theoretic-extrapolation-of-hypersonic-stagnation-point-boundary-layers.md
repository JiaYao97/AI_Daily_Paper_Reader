## 摘要
受需要将地面实验结果外推到高速飞行条件的推动，我们提出了高超声速驻点边界层的李等价对称性分析。我们展示了等价对称性在物理上与高超声速相关的一组耦合常微分方程上的应用。通过允许物性定律与自变量和因变量一起变换，在相似性约化的驻点ODE公式内导出的不变量识别出可用于将实验室尺度预测外推到飞行的非线性映射族。在实践中，实施这些映射需要实验室尺度的ODE解以及实验室和飞行侧的热化学物性数据，这些数据通常可从现有数据库中获得。这些映射是针对边界层内相似性约化的无量纲温度导出的，并显示在一系列相关情况中与独立计算的飞行解重合。

## Abstract
Motivated by the need to extrapolate the results from ground-based experiments to the conditions of high-speed flight, we present the Lie equivalence symmetry analysis of the hypersonic stagnation-point boundary layers. We demonstrate the application of the equivalence symmetry on the set of coupled ODEs which are physically relevant in hypersonics. By allowing the property laws to transform along with the independent and dependent variables, the invariants derived within the similarity-reduced stagnation-point ODE formulation, identify families of non-linear maps that can be used to extrapolate the laboratory-scale predictions to flight. In practice, implementing these maps requires the laboratory-scale ODE solution together with the lab- and flight-side thermochemical property data, which are generally available from existing databases. The maps are derived for the similarity-reduced non-dimensional temperature across the boundary layer and are shown to {collapse with independently computed flight solutions} for a range of relevant cases.

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：将Lie对称性方法扩展到 **等价对称性**（Lie equivalence symmetries）。与经典Lie对称性只变换自变量和因变量不同，等价对称性**同时变换因变量、自变量以及本构关系或物性定律**（如粘度、导热系数、比热等）。这使得整个方程类（而非单个固定方程）在不同物性条件下保持形式不变，从而建立实验室与飞行条件之间的非线性映射。
- **关键技术细节**：
    1.  **约化模型**：为便于证明概念，选取相似约化后的驻点边界层常微分方程组（ODEs）作为研究对象。
        - 连续性方程：$\frac{dV}{d\eta} + F = 0$
        - 动量方程：$V\frac{dF}{d\eta} = \frac{1}{2}(R_\rho - F^2) + \frac{d}{d\eta}\left(\ell_0\frac{dF}{d\eta}\right)$
        - 能量方程：$V\frac{dg}{d\eta} = \frac{1}{C_p}\frac{d}{d\eta}\left(\chi\frac{dg}{d\eta}\right)$
        > 其中$F, g, V$为无量纲速度、温度和法向速度；$\ell_0, \chi, R_\rho, C_p$为依赖于温度$g$的物性函数。
    2.  **对称性推导**：通过将无穷小生成元作用在方程组及其辅助方程（物性函数仅依赖于$g$）上，求解线性化对称性条件，得到等价无穷小生成元：
        $$
        X = C_0\eta \frac{\partial}{\partial \eta} + (A_0-C_0)F \frac{\partial}{\partial F} + A_0V\frac{\partial}{\partial V} + \phi_g(g)\frac{\partial}{\partial g} + 2(A_0-C_0)R_\rho\frac{\partial}{\partial R_\rho} + (A_0+C_0)\ell_0\frac{\partial}{\partial\ell_0} + \left(K_0 - \frac{d\phi_g}{dg}\right)\chi\frac{\partial}{\partial \chi} + \left(K_0-A_0-C_0-\frac{d\phi_g}{dg}\right)C_p\frac{\partial}{\partial C_p}
        $$
        > 其中$A_0, C_0, K_0$为群参数；$\phi_g(g)$是自由函数，控制温度映射。
    3.  **映射推导**：通过分析对称性产生的轨道，得到关于温度映射的积分不变量，从而构建从实验室温度$g$到飞行温度$\hat{g}$的显式映射：
        $$
        \int_{\hat{g}_{\text{interior}}}^1 \chi_F(\hat{g})\, d\hat{g} = e^{K_0} \int_{g_{\text{interior}}}^1 \chi_L(g)\, dg
        $$
        > 这里$\chi_L$和$\chi_F$分别是实验室和飞行侧的导热-密度耦合函数。该方程可简化为：$\hat{g} = H_F^{-1}\left(e^{K_0} H_L(g)\right)$，其中$H(g) = \int_g^1 \chi(s)ds$。常数$K_0$由壁面条件唯一确定：$K_0 = \ln\left(H_F(w) / H_L(w)\right)$。
    4.  **映射实现算法**：
        - 给定实验室解（$g$剖面）和实验室热化学数据（$\chi_L$）。
        - 独立获取飞行侧热化学数据（$\chi_F$），无需飞行解本身。
        - 计算$H_L(g)$和$H_F(\hat{g})$（后者在任意$g$网格上计算）。
        - 利用壁面条件确定$K_0$。
        - 对每个实验室壁面点，计算$e^{K_0} H_L(g)$，通过插值$H_F$的反函数得到对应的$\hat{g}$。
