## 摘要
尽管参数高效微调（PEFT）通常针对注意力投影器，但其在需要序列状态累积的任务中的有效性尚未充分探索。我们研究了针对此类任务的PEFT是否能从状态空间模型（SSM）适配器中获益，以及MLP块是否是更好的注入点。我们引入了Hankel降阶模型（HRM）适配器，这是一种基于SSM的残差模块，通过经验Hankel格拉姆矩阵的平衡截断初始化。利用系统矩阵$\bar{A}$的时不变性，HRM实现了基于FFT的精确并行扫描，在所有上下文长度上达到与LoRA相同的计算效率。在Mistral-7B（可训练参数840万）的等参数量评估中，HRM在LongBench任务上优于LoRA变体，包括QuALITY（相对准确率提升34.8%）和QMSum（相对ROUGE-1提升71.6%）。HRM在18种合成状态追踪（DFA、Parity）和字符级语言建模（enwik8）配置中进一步展示了持续的优越性。门控分析表明，HRM适配器有效地学会了调节循环，为长上下文序列建模提供了比低秩适应更鲁棒的架构替代方案。

## Abstract
While parameter-efficient fine-tuning (PEFT) typically targets attention projectors, its efficacy for tasks requiring sequential state accumulation remains under-explored. We examine if PEFT for such tasks can benefit from state space model (SSMs) adapters, and if MLP blocks are better injection sites. We introduce Hankel Reduced order Model (HRM) adapter, an SSM-based residual module initialized via Balanced Truncation of empirical Hankel Grammians. By leveraging the time-invariance of the system matrix $\bar{A}$, HRM enables an exact FFT-based parallel scan, achieving computational parity with LoRA across all context lengths. In iso-parametric evaluations on Mistral-7B (8.4M trainable parameters), HRM outperforms LoRA variants on LongBench tasks, including QuALITY (+34.8\% relative accuracy) and QMSum (+71.6\% relative ROUGE-1). HRM further demonstrates consistent superiority across 18 configurations of synthetic state-tracking (DFA, Parity) and character-level language modeling (enwik8). Gate analysis reveals that HRM adapters effectively learn to modulate recurrence, providing a robust architectural alternative to low-rank adaptation for long-context sequence modeling.

- **核心问题**：现有参数高效微调（PEFT）方法（如LoRA及其变体）对每个位置的输出仅依赖于当前输入 $x_t$，缺乏跨位置的记忆能力，因此无法处理需要序列状态累积的长上下文任务（如确定性有限自动机状态跟踪、长文档理解）。
- **研究动机**：探索是否能够构造一种PEFT适配器，同时满足：
    1. 为冻结的Transformer引入时域循环状态；
    2. 可被可证明地压缩到最小状态维度；
    3. 计算效率与LoRA等价。
- **整体含义**：提出一种基于状态空间模型（SSM）的残差适配器——Hankel降阶模型（HRM）适配器，将其并行注入到预训练Transformer的MLP块中，从而在不破坏注意力机制的前提下提供循环记忆能力。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- **HRM适配器**：在冻结Transformer的每个MLP块旁并联一个时不变线性SSM残差模块，其输出 $y_t$ 通过门控 $\alpha$ 加到MLP输出上。
- **关键公式**：$$h_t^{\text{out}} = h_t^{\text{MLP}} + \alpha \cdot y_t, \quad y_t = C s_t, \quad s_t = \bar{A} s_{t-1} + \bar{B} h_t^{\text{MLP}}$$
- **模型降阶**：利用控制理论中的**平衡截断**（Balanced Truncation）对SSM进行可证明的降维，基于可控性格拉姆矩阵 $W_c$ 和可观测性格拉姆矩阵 $W_o$ 的乘积的Hankel奇异值（HSV）进行截断，并满足Glover $H_\infty$ 误差界。

### 关键技术细节
- **时不变 $\bar{A}$ 的强制稳定参数化**：$\bar{A} = \operatorname{diag}(\bar{a}_1,\dots,\bar{a}_d)$，其中 $\bar{a}_i = \exp(-\exp(\log A_i))$，保证 $0<\bar{a}_i<1$，系统无条件稳定。
- **FFT并行扫描**：因为 $\bar{A}$ 时不变，HRM的循环等价于因果线性卷积，可通过FFT在 $O(T\log T)$ 内完成前向计算，与LoRA的计算时间完全一致（附录B表2验证）。
- **注入位置选择**：适配器并联在MLP块，而非注意力块。理由：1) 注意力已提供跨位置全局上下文；2) MLP是位置无关计算，加入循环可补充缺失的时序依赖；3) 并联设计避免干扰冻结MLP的梯度流。
- **门控初始化**：门控标量 $\alpha$ 初始化为小值（如0.1），避免训练初期干扰预训练行为；实验显示若初始化为1.0会导致发散。
- **Hankel奇异值（HSV）**：用于表征任务的内存需求复杂度，通过平衡截断自动决定保留的状态维度 $\hat{d}$。

### 公式或算法流程（文字说明）
1. 初始化SSM参数 $\bar{A}$, $\bar{B}$, $C$, $\alpha$；
2. 对输入序列 $\{h_t^{\text{MLP}}\}$，利用FFT计算零填充后的卷积得到输出序列 $\{y_t\}$；
3. 计算 $\alpha \cdot y_t$ 并加到原始MLP输出上；
4. 在训练若干轮后，收集状态轨迹，计算经验格拉姆矩阵，求得HSV，进行平衡截断，压缩SSM状态维度；
5. 继续训练截断后的HRM适配器（或直接使用截断后的模型进行推理）。

## 7. 优点

- **理论可解释性**：基于控制论的平衡截断提供有界误差（Glover界），压缩具有可证明的保证，而非启发式。
- **计算效率**：利用时不变$\bar{A}$实现FFT并行扫描，在所有上下文长度上与LoRA计算时间完全相同（附录B表2），避免了循环调用的Python开销。
- **注入位置创新**：选择MLP而非注意力块，并通过实验和论证表明此位置更适合加入循环记忆，同时保持注意力机制不受干扰。
- **广泛实验验证**：覆盖合成任务、语言建模、长文档理解等多种场景，并进行了多个消融实验，验证了算法各组件的作用。
- **Hankel奇异值的实用价值**：提供了一种评估任务内存需求的新度量，有助于指导适配器参数分配。
