## 摘要
基于经典本征正交分解（POD）的混沌流伽辽金投影模型通常需要大量模态以及稳定化或闭合项才能达到足够的精度和长期稳定性。我们提出了一种新颖的可微编程框架，该框架在不增加模态数量或引入额外闭合项的情况下稳定低秩POD-伽辽金模型，从而同时实现高效益和高精度。模型稳定化通过使用可微编程调整POD-伽辽金中的线性张量和二次张量实现，并在短期轨迹数据上进行训练。本研究的一个关键发现是，对于混沌系统，纯粹的逐点轨迹损失函数会导致较差的长期准确性。相比之下，结合轨迹误差和基于物理的能量守恒项的混合损失函数提供了优越的长期性能。我们在雷诺数Re = 30,000的混沌方腔驱动流动上演示了该方法。与经典POD-伽辽金方法相比，稳定化ROM的计算成本降低了一个数量级：仅使用20个模态即可保持准确和稳定，而经典ROM需要80个POD模态。

## Abstract
Classical Proper Orthogonal Decomposition (POD)-based Galerkin projection models of chaotic flows typically require a large number of modes as well as stabilization or closure terms to achieve adequate accuracy and long-term stability. We present a novel differentiable programming framework that stabilizes low-rank POD-Galerkin models without increasing the number of modes or introducing additional closure terms, thereby delivering both high efficiency and high accuracy. Model stabilization is achieved by tuning the linear and quadratic tensors in the POD-Galerkin using differentiable programming, trained on short-term trajectory data. A key finding of this study is that a purely point-wise trajectory-based loss function yields poor long-term accuracy for chaotic systems. In contrast, a hybrid loss function that combines trajectory error with a physics-based conservation-of-energy term provides superior long-term performance. We demonstrate the approach on a chaotic lid-driven cavity flow at Re = 30,000. The stabilized ROM achieves an order-of-magnitude reduction in computational cost compared with the classical POD-Galerkin method: it remains accurate and stable with only 20 modes, whereas the classical ROM requires 80 POD modes.

## 2. 论文提出的方法论

### 2.1 核心思想
- 利用可微编程（Differentiable Programming）直接学习POD-Galerkin ODE中的线性项（$L_{ij}$）和常数项（$C_i$），保持二次项（$Q_{ijk}$）不变，从而保留底层物理结构的同时补偿截断效应。
- 损失函数设计：提出**混合损失**，结合轨迹误差（点态MSE）和基于物理学的时间平均湍动能守恒惩罚项，平衡短期准确性和长期稳定性。

### 2.2 关键技术细节
- **GP-ROM ODE**：
  $$
  \dot{a}_i = C_i + \sum_{j=1}^{N} L_{ij} a_j + \sum_{j=1}^{N} \sum_{k=1}^{N} Q_{ijk} a_j a_k
  $$
  其中 $C_i, L_{ij}, Q_{ijk}$ 通过伽辽金投影从DNS快照计算得到。
- **DPG-ROM ODE**：将 $C_i$ 和 $L_{ij}$ 替换为可训练参数 $C_i^p, L_{ij}^p$，即
  $$
  \dot{a}_i = C_i^p + \sum_{j=1}^{N} L_{ij}^p a_j + \sum_{j=1}^{N} \sum_{k=1}^{N} Q_{ijk} a_j a_k
  $$
  初始化使用原始GP-ROM系数。
- **损失函数**：
  - 轨迹损失：$L_{\text{traj}} = \frac{1}{N T} \sum_{n=1}^{T} \sum_{i=1}^{N} \left( \frac{a_{i,n}^{\text{pred}} - a_{i,n}^{\text{ref}}}{\sigma} \right)^2$
  - 能量损失：$L_{\text{energy}} = \left( \frac{\bar{K}^{\text{pred}} - \bar{K}^{\text{ref}}}{\bar{K}^{\text{ref}}} \right)^2$，其中 $\bar{K}$ 为时间平均湍动能。
  - 总损失：$L_{\text{total}} = \lambda L_{\text{traj}} + (1-\lambda) L_{\text{energy}}$，本文通过扫描确定最优 $\lambda = 0.5$。
- **训练流程**：在短训练窗口（25个非时间单位，2500个快照）内正向积分ODE，使用伴随灵敏度方法和反向模式自动微分（AD）计算梯度，Adam优化器（学习率 $1\times10^{-3}$ 或 $1\times10^{-4}$）。引入验证集（40%数据）和早停机制防止过拟合。参数归一化到 $[-1,1]$ 以改善梯度流动。
- **能量成分分析**：通过时间平均的湍动能平衡方程 $\partial \bar{K}/\partial t \approx \bar{P} + \bar{D} + \bar{C} + \bar{T} = 0$ 评估生产项 $\bar{P}$ 和耗散项 $\bar{D}$，判断ROM是否学习到物理一致的耗散机制。

## 6. 论文的主要结论与发现

1. **DPG-ROM可微编程框架成功稳定低截断GP-ROM**：仅20模态的DPG-ROM（混合损失）可获得与80模态经典GP-ROM相当的精度和稳定性，计算成本降低一个数量级。
2. **纯轨迹损失局限性**：$\lambda=1$ 的模型在训练窗口内准确性好，但长期稳定性不足，出现能量尖峰和系数漂移，说明仅优化点态误差无法保证物理一致的耗散机制。
3. **混合损失优势**：$\lambda=0.5$ 的混合损失提供了最优长期性能，平衡了短期跟踪和能量守恒，学习到更合理的耗散机制，TKE演变与DNS吻合更好，相空间分布更紧致。
4. **校准方法不足**：校准GP-ROM在高度截断情况下效果差，甚至比未校准模型更差，其线性化误差导致累积误差放大。
5. **模态数影响**：随模态数增加（20→30），DPG-ROM之间的差异减小，能量惩罚的增益变弱；80模态下所有方法都已稳定，DPG-ROM的优势在高模态时不再显著。

## 7. 优点

- **方法创新**：提出可微编程+混合损失的ROM稳定框架，结构简单（不增加模态或闭合项），同时保留ODE物理形式，增强了可解释性。

## 8. 不足与局限

- **泛化性有限**：所有实验仅在二维方腔驱动流（Re=30,000）上进行，未验证其他雷诺数、几何形状或三维流动，结论能否推广到更广泛混沌流需进一步验证。
