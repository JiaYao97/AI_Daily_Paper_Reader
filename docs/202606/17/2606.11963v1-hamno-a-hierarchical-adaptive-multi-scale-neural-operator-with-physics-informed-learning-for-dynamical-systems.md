## 摘要
神经算子提供了一个强大的框架，用于直接在函数空间中学习偏微分方程的解映射。然而，许多现有架构仍然难以表示涉及多尺度结构、长程相互作用和稳定长时间演化的非线性时变系统。在这项工作中，我们引入了分层自适应多尺度神经算子（HAMNO），这是一种结合局部卷积表示、全局谱算子和分层编码器-解码器处理的神经算子架构。HAMNO的核心是一个数据依赖门控机制，它在每个空间位置自适应地平衡局部和全局信息，使模型能够解析细尺度特征同时保留长程依赖性。我们进一步开发了物理信息扩展PI-HAMNO，基于多目标损失策略，结合数据拟合与强形式和弱形式物理约束。强形式项惩罚物理坐标中域积分平方偏微分方程残差，而弱形式项通过将控制残差乘以有限元测试函数，并使用基于质心的四面体求积评估所得单元积分来构建。该框架在定义于立方域上的非周期Allen-Cahn（AC）、Cahn-Hilliard（CH）和Swift-Hohenberg（SH）方程上进行了评估。在长期展开、数据有限训练、分布外初始条件偏移和随机种子变化下，HAMNO相比标准神经算子基线提高了预测精度，而PI-HAMNO进一步增强了稳定性、物理一致性和数据效率。该实现公开于https://github.com/MBamdad/HAMNO。

## Abstract
Neural operators provide a powerful framework for learning solution mappings of partial differential equations directly in function space. However, many existing architectures still struggle to represent nonlinear time-dependent systems that involve multi-scale structures, long-range interactions, and stable long-time evolution. In this work, we introduce the Hierarchical Adaptive Multi-scale Neural Operator (HAMNO), a neural-operator architecture that combines local convolutional representations, global spectral operators, and hierarchical encoder-decoder processing. The central component of HAMNO is a data-dependent gating mechanism that adaptively balances local and global information at each spatial location, allowing the model to resolve fine-scale features while preserving long-range dependencies.   We further develop a physics-informed extension, PI-HAMNO, based on a multi-objective loss strategy that combines data fitting with strong- and weak-form physics constraints. The strong-form term penalizes the domain-integrated squared PDE residual in physical coordinates, while the weak-form term is constructed by multiplying the governing residual by finite-element test functions and evaluating the resulting element integrals using centroid-based tetrahedral quadrature. The framework is evaluated on non-periodic Allen-Cahn (AC), Cahn-Hilliard (CH), and Swift-Hohenberg (SH) equations defined on cubic domains. Across long-horizon rollout, data-limited training, out-of-distribution initial-condition shifts, and random-seed variations, HAMNO improves predictive accuracy over standard neural-operator baselines, while PI-HAMNO further enhances stability, physical consistency, and data efficiency. The implementation is publicly available at https://github.com/MBamdad/HAMNO .

### 论文提出的方法论：核心思想与关键技术细节

论文提出了一种名为**层次化自适应多尺度神经算子（HAMNO）** 及其物理信息扩展（PI-HAMNO）的方法。

1.  **核心思想**：
    -   **自适应平衡**：核心创新在于一个**数据依赖的门控机制**，该机制允许模型在每一个空间位置，根据输入特征动态地、自适应地平衡局部信息（来自卷积操作）和全局信息（来自谱域操作）。
    -   **多尺度层次化**：采用编码器-解码器结构，通过下采样（缩放）和上采样（放大）的路径，在不同空间分辨率上处理信息，并通过跳跃连接保留细粒度特征。这实现了从粗到细的多尺度特征学习。
    -   **物理约束正则化**：通过引入**强形式**和**弱形式**两种互补的物理残差作为损失函数，将偏微分方程（PDE）的物理知识直接融入训练过程，以提升预测的物理一致性、稳定性和数据效率。

2.  **关键技术细节**：
    -   **HAMNO 架构**：
        -   **输入提升**：将输入的时间序列状态和空间坐标通过一个学习到的线性层映射到高维潜空间。
        -   **局部-全局算子分解**：在每个算子块中，首先对潜特征进行归一化，然后分别通过局部卷积算子 $K_{local}$ 和全局谱算子 $K_{global}$（在傅里叶域中实现）进行处理。
        -   **自适应门控融合**：通过数据依赖的权重 $\alpha(x)$ 和 $\beta(x)$ 对局部和全局信息进行加权融合：$h_{fused}(x) = \alpha(x) h_{local}(x) + \beta(x) h_{global}(x)$。
        -   **残差更新**：融合后的特征经过通道混合和两次残差连接（包括一次特征融合后的加法和一次非线性变换后的加法），以优化稳定性和深层学习能力。
        -   **层次化编解码**：编码阶段逐步下采样以捕捉大尺度全局信息；解码阶段上采样并结合跳跃连接恢复细节；最后通过一个输出头投影得到预测状态。

    -   **PI-HAMNO 物理约束**：
        -   **多目标损失函数**：总损失为 $L_{total} = (1-\lambda)L_{data} + \lambda L_{phys}$，其中 $\lambda$ 控制数据拟合与物理约束的权重。
        -   **强形式残差** $L_{strong}$：直接使用有限差分近似计算PDE在网格点上的残差（如 $R_{FD} = \frac{u^{n+1}_\theta - u^n}{\Delta t} - \mathcal{N}(u^{n+1}_\theta)$），并对该残差在整个计算域上进行积分（通过分解为四面体单元并在质心处求积）。
        -   **弱形式残差** $L_{weak}$：将PDE残差乘以有限元测试函数（如伽辽金法中的 $\text{P1}$ 形函数），并在四面体单元上进行积分。通过分部积分，降低了导数的阶数，并自然地处理了诺伊曼边界条件。其形式如 $r_{K,i} = \int_K u_t N_i dV + \int_K (\text{derivative terms}) dV + \int_K (\text{reaction terms}) dV$。最终的弱形式损失是所有单元和局部测试函数残差平方和。

### 论文的主要结论与发现

1.  **HAMNO**：相比所有基线方法，在长期滚动预测、数据有限训练和分布外泛化方面，均展现出更优越的预测精度和更稳定的误差控制能力。这归因于其自适应平衡局部与全局信息的机制和层次化多尺度结构。
2.  **PI-HAMNO**：通过引入多目标强/弱形式物理约束，进一步提升了模型在预测稳定性、物理一致性（如能量耗散、质量守恒）和数据效率（特别是在数据稀缺时）方面的性能。

4.  **鲁棒性**：在多个随机种子下的实验表明，HAMNO和PI-HAMNO的性能提升是稳定的，不是由偶然的有利初始化或数据分割造成的。
