## 摘要
神经算子为直接在函数空间学习偏微分方程的解映射提供了强大框架。然而，许多现有架构仍难以表示涉及多尺度结构、长程相互作用和稳定长时间演化的非线性时变系统。本文提出层级自适应多尺度神经算子（HAMNO），一种结合局部卷积表示、全局谱算子和层级编码器-解码器处理的神经算子架构。HAMNO的核心是一种依赖于数据的门控机制，可在每个空间位置自适应平衡局部与全局信息，使模型在保持长程依赖的同时解析细尺度特征。我们进一步基于多目标损失策略开发了物理信息扩展版PI-HAMNO，该策略结合了数据拟合、强形式与弱形式物理约束。强形式项惩罚物理坐标中域积分平方PDE残差，弱形式项通过将控制残差乘以有限元测试函数并使用基于质心的四面体求积法计算所得单元积分来构建。该框架在立方域上的非周期Allen-Cahn、Cahn-Hilliard和Swift-Hohenberg方程上进行了评估。在长程滚动、数据有限训练、分布外初始条件偏移和随机种子变化等场景下，HAMNO相比标准神经算子基线提高了预测精度，而PI-HAMNO进一步增强了稳定性、物理一致性和数据效率。实现代码公开于https://github.com/MBamdad/HAMNO。

## Abstract
Neural operators provide a powerful framework for learning solution mappings of partial differential equations directly in function space. However, many existing architectures still struggle to represent nonlinear time-dependent systems that involve multi-scale structures, long-range interactions, and stable long-time evolution. In this work, we introduce the Hierarchical Adaptive Multi-scale Neural Operator (HAMNO), a neural-operator architecture that combines local convolutional representations, global spectral operators, and hierarchical encoder-decoder processing. The central component of HAMNO is a data-dependent gating mechanism that adaptively balances local and global information at each spatial location, allowing the model to resolve fine-scale features while preserving long-range dependencies.   We further develop a physics-informed extension, PI-HAMNO, based on a multi-objective loss strategy that combines data fitting with strong- and weak-form physics constraints. The strong-form term penalizes the domain-integrated squared PDE residual in physical coordinates, while the weak-form term is constructed by multiplying the governing residual by finite-element test functions and evaluating the resulting element integrals using centroid-based tetrahedral quadrature. The framework is evaluated on non-periodic Allen-Cahn (AC), Cahn-Hilliard (CH), and Swift-Hohenberg (SH) equations defined on cubic domains. Across long-horizon rollout, data-limited training, out-of-distribution initial-condition shifts, and random-seed variations, HAMNO improves predictive accuracy over standard neural-operator baselines, while PI-HAMNO further enhances stability, physical consistency, and data efficiency. The implementation is publicly available at https://github.com/MBamdad/HAMNO .

### 2. 方法论
- **核心思想**：提出 **HAMNO**（Hierarchical Adaptive Multi-scale Neural Operator），通过数据依赖的门控机制在每个空间位置自适应平衡局部卷积与全局谱信息，并利用层级编码器–解码器结构实现多尺度表示。进一步引入 **PI-HAMNO**，在损失函数中同时融入强形式（有限差分域积分残差）和弱形式（有限元测试函数+四面体质心求积）物理约束。
- **关键技术细节**：
  - **局部–全局算子分解**：对归一化后的隐特征分别应用局部卷积 $h_{\text{local}} = \mathcal{K}_{\text{local}}(h)$ 和全局傅里叶谱算子 $h_{\text{global}} = \mathcal{K}_{\text{global}}(h)$。
  - **自适应门控融合**：$h_{\text{fused}}(x) = \alpha(x) h_{\text{local}}(x) + \beta(x) h_{\text{global}}(x)$，其中 $\alpha(x), \beta(x)$ 从输入特征学习得到，实现数据驱动的局部–全局平衡。
  - **残差更新**：每个 HAMNO 块包含两步残差：$v' = v + \tilde{h}_{\text{fused}}$，$v_{\ell+1} = v' + \Psi_{\ell}(\text{Norm}(v'))$，其中 $\Psi_{\ell}$ 为逐点通道混合。
  - **层级结构**：编码器逐步下采样（zoom-out），解码器通过上采样与跳跃连接恢复细节（zoom-in），最终与初始提升特征融合。
  - **PI-HAMNO 损失**：$\mathcal{L}_{\text{total}} = (1-\lambda)\mathcal{L}_{\text{data}} + \lambda (\mathcal{L}_{\text{strong}} + \mathcal{L}_{\text{weak}})$。
    - 强形式：$\mathcal{L}_{\text{strong}} \approx \sum_K (R_K^c)^2 |K|$，$R_K^c$ 为网格单元四面体质心处的有限差分残差。
    - 弱形式：$\mathcal{L}_{\text{weak}} = \frac{1}{N_e}\sum_K \sum_i (r_{K,i})^2$，$r_{K,i}$ 为单元 $K$ 上测试函数 $N_i$ 对应的 Galerkin 残差，采用 $P_1$ 线性形状函数和四面体质心求积。
  - **特化**：对 Allen–Cahn、Cahn–Hilliard（分裂为 $u,\mu$ 耦合系统）和 Swift–Hohenberg（引入辅助变量 $p=\Delta u$ 及 Helmholtz 型一致性条件）分别给出具体强/弱形式。

### 3. 实验设计
- **数据集**：三维立方域上的 Allen–Cahn（AC）、Cahn–Hilliard（CH）、Swift–Hohenberg（SH）方程，均采用均匀网格 $32\times32\times32$，非周期 homogeneous Neumann 边界条件。每个方程共 600 条轨迹，训练取 50/100/200 条，固定测试集 50 条。初始场由高斯随机场生成（参数 $\alpha,\tau$ 控制相关长度）。
- **基准方法**：FNO、F-FNO、DeepONet、U-NO、U-Net、U-FNO。
- **评价指标**：自回归 rollout 下的相对 $L_2$ 误差 $\frac{\|u_{\text{pred}} - u_{\text{true}}\|_{L_2}}{\|u_{\text{true}}\|_{L_2}}$。
- **实验场景**：
  - 分布内（ID）：不同训练样本数（50/100/200）和物理权重 $\lambda$（0, 0.25, 1）。
  - 分布外（OOD）：改变初始场参数 $(\alpha,\tau)$ 至未见值（两个设置）。

  - 鲁棒性：4 个随机种子（0,36,123,1024）和 10 个随机种子（详尽列表）重复实验。

### 6. 主要结论与发现
- **HAMNO** 在长时 rollout 中一致优于所有基线，尤其在 CH（四阶守恒）和 SH（图形成型）等更刚性的动力学中优势显著。
- **PI-HAMNO** 进一步降低累积误差，提高物理一致性（如质量守恒、自由能下降），并在数据有限（$\lambda=0.25$）时表现最佳。
- **消融实验**证明强+弱形式联合优于单一形式：强形式提供局域 PDE 约束，弱形式提供变分正则化，两者互补。
- **OOD 测试**显示 PI-HAMNO 在未见初始场分布下仍保持最高精度，说明其泛化能力强。
- **种子鲁棒性**表明 HAMNO/PI-HAMNO 的性能随机初始化波动较小（标准差与均值比大多 <0.5），相较于部分基线更稳定。

### 7. 优点
- **架构创新**：自适应门控融合局部与全局信息，比固定加法融合（如 U-FNO）更灵活。
- **物理约束设计**：首次将强形式（有限差分域积分）与弱形式（有限元 Galerkin 残差）结合用于神经算子，兼顾点精度与全局变分一致性。
- **全面实验**：覆盖 ID、OOD、消融、多种子，且详尽报告了每条轨迹的误差和计算时间，结果可复现。
- **开源实现**：代码公开，促进后续研究。

### 8. 不足与局限
- **应用范围**：仅评估了三维规则立方域上的均匀网格。未测试非结构化网格、复杂几何或更高维问题。
- **计算开销**：HAMNO 的训练/推理时间通常高于 FNO 等简单基线（约 2–5 倍），PI-HAMNO 因需要计算物理残差而更高。
- **弱形式实现**：依赖四面体单元有针对性网格划分和质心求积，对于非均匀或自适应网格可能复杂，且 $P_1$ 线性元精度有限。
- **超参数敏感性**：物理权重 $\lambda$ 对性能有影响（文中仅测试 0, 0.25, 1），但未系统分析 $\lambda$ 的连续变化。
- **诊断指标**：仅报告相对 $L_2$ 误差和部分能量/质量，未涉及频谱分析、物理量（如界面位置精度）等更细粒度评估。
