## 摘要
高保真计算流体动力学（CFD）对车辆气动分析至关重要，但其成本仍制约着早期设计探索。基于机器学习的表面场预测提供了一种更快速的替代方案，前提是模型能够高效捕捉全局流场上下文和局部几何细节。本文提出一种基于机器学习的方法，称为几何感知三平面场网络（GTF-Net），用于车辆气动压力和壁面剪切应力预测。GTF-Net通过共享多层感知器（MLP）和平滑双线性光栅化直接从采样表面点构建三平面特征。然后，这些平面由双流主干处理，该主干结合了自适应傅里叶神经算子（AFNO）频谱混合与卷积神经网络（CNN）细化，从而在同一表示中对远程气动耦合和局部几何引起的变异进行建模。在查询阶段，采样的三平面特征与车辆对齐方向坐标、法向投影特征以及基于体素的曲率代理相结合。GTF-Net与Transolver、几何信息神经算子（GINO）以及基于三平面的代理模型TripNet进行了比较。GTF-Net将压力预测的相对L2误差从最强基线的0.157降低到0.145，壁面剪切应力预测从0.237降低到0.226。消融实验结果表明，AFNO混合、局部CNN细化和查询端几何编码均有助于提高精度，支持了将结构化三平面表示与显式气动几何线索相结合的所提机制。

## Abstract
High-fidelity computational fluid dynamics (CFD) is crucial to vehicle aerodynamic analysis, but its cost still constrains early-stage design exploration. Machine-learning-based surface-field prediction offers a faster alternative if the model can efficiently capture both global flow context and local geometric detail. This work proposes a machine-learning-based method, named the geometry-aware triplane field network (GTF-Net), for vehicle aerodynamic pressure and wall shear stress prediction. GTF-Net constructs triplane features directly from sampled surface points through a shared multilayer perceptron (MLP) and smooth bilinear rasterization. The planes are then processed by a dual-stream backbone that combines adaptive Fourier neural operator (AFNO) spectral mixing with convolutional neural network (CNN) refinement, so long-range aerodynamic coupling and local geometry-induced variations are modeled in the same representation. At query stage, sampled triplane features are combined with vehicle-aligned directional coordinates, normal-projection features, and a voxel-based curvature proxy. GTF-Net is compared with Transolver, geometry-informed neural operator (GINO), and TripNet, a triplane-based surrogate model. GTF-Net improves the relative L2 error from the strongest baseline value of 0.157 to 0.145 for pressure prediction and from 0.237 to 0.226 for wall shear stress prediction. Ablation results show that AFNO mixing, local CNN refinement, and query-side geometric encoding each contribute to accuracy, supporting the proposed mechanism of combining structured triplane representation with explicit aerodynamic geometry cues.

### 2. 论文提出的方法论（核心思想与关键技术细节）

本文提出了一种名为**几何感知三平面场网络（GTF-Net）** 的模型，其核心思想是构建一个端到端的三平面场网络，并在解码阶段显式注入几何信息。

*   **核心流程**：输入车辆表面点云 -> 构建三平面特征 -> 双流骨干网络处理 -> 几何感知查询解码 -> 输出预测场。

*   **关键技术细节**：
    1.  **三平面特征构建**：
        *   输入：表面点坐标集合 $X = \{x_i\}_{i=1}^N$。
        *   点特征提升：使用一个**共享多层感知器（Shared MLP）** 将每个表面点 $x_i$ 映射到一个高维特征向量 $h_i$：$$h_i = E_\theta(x_i), \quad h_i \in \mathbb{R}^{d_e}$$
        *   光栅化：将点特征通过**平滑双线性光栅化**投影到三个轴对齐的平面（xy, yz, xz）上。对于平面上的某个网格位置 $\xi$，其特征为：$$T_\pi(\xi) = \frac{\sum_{i=1}^N w_{i,\pi}(\xi)h_i}{\sum_{i=1}^N w_{i,\pi}(\xi) + \epsilon}$$，其中 $w_{i,\pi}$ 是双线性插值权重。此过程无需针对每个对象进行单独的拟合步骤。
        *   输出：三个平面最终拼接成一个三维张量 $T \in \mathbb{R}^{3d_e \times H \times W}$。

    2.  **双流骨干网络**：
        *   **目的**：同时建模全局气动相关性和局部空间模式。
        *   **AFNO 流（谱流）**：处理 $T$，使用**自适应傅里叶神经算子（AFNO）** 进行谱混合，高效捕捉长程气动耦合信息。其核心操作是：$$A_\ell(Z) = \mathcal{F}^{-1}(W_\ell(\mathcal{F}(Z)))$$
        *   **CNN 流（局部流）**：并行处理 $T$，使用残差卷积神经网络（CNN）对局部空间细节进行细化，尤其关注几何突变区域。
        *   **融合**：两个流的输出通过逐元素相加进行融合：$$T’ = T_{\text{AFNO}} + T_{\text{CNN}}$$

    3.  **几何感知查询解码**：
        *   **输入**：对于待预测的查询点 $q_j$，提取其双流处理后的三平面特征 $s(q_j)$，以及其位置编码 $\phi_{\text{pos}}(q_j)$。
        *   **几何编码 $\phi_{\text{geo}}(q_j)$**：这是本文的核心改进之一，由三个部分组成：
            *   **车辆对齐方向特征**：基于车辆对称轴和来流方向构建的笛卡尔坐标系下的相对位置和方向。
            *   **法向投影特征**：利用查询点的表面法向量在车辆坐标系中的投影信息。
            *   **曲率代理**：通过局部体素协方差矩阵计算的特征值比率来近似局部曲率：$$\kappa(q_j) = \frac{\lambda_1}{\lambda_1 + \lambda_2 + \lambda_3}$$，其中 $\lambda_1 \le \lambda_2 \le \lambda_3$ 为协方差矩阵的特征值。
        *   **融合与预测**：位置编码与几何编码通过一个可学习的残差门控机制融合：$$z(q_j) = \phi_{\text{pos}}(q_j) + \tanh(\alpha)\phi_{\text{geo}}(q_j)$$，其中 $\alpha$ 初始化为 0。最后，将三平面特征和融合后的编码输入一个多层感知器（MLP）得到最终预测值 $\hat{u}_\theta(q_j; X)$。
