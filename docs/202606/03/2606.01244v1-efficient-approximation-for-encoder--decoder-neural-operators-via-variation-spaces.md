## 摘要
我们研究使用编码器-解码器神经网络的算子学习。受神经网络函数空间理论的启发，我们引入了一个变分空间作为非线性算子的无穷维结构类。该空间通过直接在输入和输出空间上的向量值测度来定义。针对该空间中的算子，我们建立了编码器-解码器两层网络在Bochner $L^q$ 范数下的逼近界。所得的误差界分解为输入编码误差、输出编码误差和一个阶为 $N^{-1/2}$ 的有限宽逼近项，且常数与输入和输出编码维度无关。当输入和输出编码误差在编码维度上呈多项式衰减时，这些估计给出了代数逼近和学习速率。该结果为超越一般Lipschitz或Fréchet可微算子类的高效神经算子学习提供了理论保证。

## Abstract
We study operator learning using encoder--decoder neural networks. Inspired by the function-space theory of neural networks, we introduce a variation space as an infinite-dimensional structural class for nonlinear operators. This space is defined through vector-valued measures directly on the input and output spaces. For operators in this space, we establish approximation bounds for encoder--decoder two-layer networks in the Bochner $L^q$ norm. The resulting error bound decomposes into the input encoding error, the output encoding error, and a finite-width approximation term of order $N^{-1/2}$, with a constant independent of the input and output encoding dimensions. When the input and output encoding errors decay polynomially in the encoding dimensions, these estimates yield algebraic approximation and learning rates. The results provide an theoretical guarantees for efficient neural operator learning beyond general Lipschitz or Fréchet differentiable operator classes.

### 2.1 核心思想

- **变分空间定义**：将有限维变分空间（通过参数空间上的有限测度表示两层神经网络）推广到向量值算子。具体地，定义算子 $G^\dagger: U \to V$ 属于变分空间 $\mathcal{V}(U; V)$，若存在 $U$ 的增广空间 $\overline{U} = \mathbb{R} \oplus U$ 上的有限 $V$ 值 Borel 测度 $\Lambda$，使得
  $$
  G(u) = \int_{\overline{B}_U} \sigma(\langle \overline{u}, \overline{h} \rangle_{\overline{U}}) \, d\Lambda(\overline{h}), \quad u \in U.
  $$
  其中 $\sigma$ 为 Lipschitz 激活函数，$\overline{B}_U$ 是 $\overline{U}$ 的闭单位球（赋弱拓扑使之紧且可度量）。变分范数定义为 $\|G\|_{\mathcal{V}} = \inf |\Lambda|(\overline{B}_U)$。

- **与已有工作的区别**：相较于 Korolev (2022) 的 Banach 值变分空间（要求输出空间为 Riesz 空间且正部操作弱*连续，在 $L^q([0,1]^d)$ 中不成立），本文不要求输出空间的格结构，且在强 Bochner 范数下逼近，并显式量化了编码误差和有限宽误差。

### 2.2 关键技术细节

- **编码器-解码器架构**：给定 $U$ 和 $V$ 的标准正交基 $\{\phi_i\}$, $\{\psi_j\}$，定义
  $$
  E_m^U u = (\langle u,\phi_1\rangle, \dots, \langle u,\phi_m\rangle), \quad D_n^V y = \sum_{j=1}^n y_j \psi_j,
  $$
  投影算子 $P_m^U = D_m^U E_m^U$, $P_n^V = D_n^V E_n^V$。待学习的有限维映射 $g: \mathbb{R}^m \to \mathbb{R}^n$ 为两层神经网络：
  $$
  g(x) = \sum_{k=1}^N b_k \sigma(\alpha_k + a_k \cdot x), \quad \alpha_k^2 + |a_k|^2 \le 1, \; \sum_{k=1}^N |b_k| \le B.
  $$

- **逼近误差分解**：对目标 $G^\dagger \in \mathcal{V}$，本文定理 3.1 证明了存在 $g_{m,n,N}$ 使得
  $$
  \|G^\dagger - D_n^V \circ g_{m,n,N} \circ E_m^U\|_{L^q_\rho(V)} \le \underbrace{\|(I-P_n^V)G^\dagger\|_{L^q_\rho(V)}}_{\text{输出编码误差}} + \left( L_\sigma \underbrace{\|I-P_m^U\|_{L^q_\rho(U)}}_{\text{输入编码误差}} + \frac{C_q(|\sigma(0)| + L_\sigma \overline{M}_q)}{\sqrt{N}} \right) \|G^\dagger\|_{\mathcal{V}}.

  其中 $C_q$ 仅依赖于 $q$，$N$ 为隐层宽度，$m,n$ 为编码维数。误差分解明确展示了三类贡献：输出投影残差、输入投影残差（由基逼近性决定）、以及有限宽逼近项（阶 $N^{-1/2}$，常数与 $m,n$ 无关）。

- **证明概要**：利用向量值测度的极分解，将 $G^\dagger$ 写为关于概率测度 $\nu = |\Lambda|/V_\Lambda$ 的积分；然后通过独立同分布采样构造蒙特卡洛近似，利用 Hilbert 空间中的 Marcinkiewicz–Zygmund 型不等式（附录 A）控制采样误差，最终得到存在性结果。构造中 $g_{m,n,N}$ 的权重 $b_k$ 由 $V_\Lambda/N$ 乘以 $E_n^V \lambda(\overline{h}_k)$ 给出。
