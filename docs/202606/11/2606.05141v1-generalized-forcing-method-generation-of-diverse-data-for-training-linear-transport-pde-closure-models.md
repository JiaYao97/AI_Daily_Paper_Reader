## 摘要
数据驱动的输运偏微分方程闭包建模需要训练数据准确、经济、多样化，并直接针对目标闭包场。我们开发了广义强迫方法（GFM），一种用于训练线性输运闭包模型的数据生成框架。GFM通过运行零初始条件且与约化动力学兼容的额外体力的模拟来生成此类数据。该框架产生了隐式GFM（iGFM），它预设了求解轨迹，以及显式GFM（eGFM），它构建了可容许强迫的基。我们将eGFM应用于三个线性输运闭包问题：均匀剪切流、空间非均匀流以及具有随机系数的均匀剪切流。结果表明，当约化变量和模型形式与底层闭包关系一致时，eGFM可以识别出准确且稳定的约化模型。

## Abstract
Data-driven closure modeling for transport partial differential equations requires training data that are accurate, affordable, diverse, and directly tailored to the target closure fields. We develop the Generalized Forcing Method (GFM), a data-generation framework for training linear transport closure models. GFM generates such data by running simulations with a zero initial condition and an extra body force that is constructed compatibly with the reduced dynamics. This framework leads to implicit GFM (iGFM), which prescribes resolved trajectories, and explicit GFM (eGFM), which constructs a basis of admissible forcings. We apply eGFM to three linear transport closure problems: homogeneous shear flows, spatially inhomogeneous flows, and homogeneous shear flows with random coefficients. The results show that eGFM can identify accurate and stable reduced models when the reduced variables and model form are consistent with the underlying closure relation.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- **解析–未解析子空间分解**：选取一组线性独立的权重函数 $\{\phi_i\}$ 定义约化变量 $f_i = \langle \phi_i, c \rangle_Y$（$Y$ 方向平均），并构造对偶基 $\{\psi_i\}$ 和投影算子 $P_r = M^+ M$（解析子空间）、$P_u = I - P_r$（未解析子空间）。
- **隔离闭包映射**：分析强迫系统 $ \partial c / \partial t = L c + s $ 的 Duhamel 解，发现目标闭包关系受未解析强迫和初始条件影响。GFM 原则：**零初始条件**（$c_0 \equiv 0$）且**强迫限制在解析子空间**（$s_u \equiv 0$），从而获得 $f$ 与闭包项 $g$ 之间的精确线性映射。

### 两种实现
- **隐式 GFM (iGFM)**：直接预设约化变量的轨迹 $f(x,t)$（例如空间余弦组合），通过 $c_r = M^+ f$ 重构解析场，再反解所需的强迫 $s = \partial c / \partial t - L c$。单次模拟即可提供训练数据，但依赖时间导数计算。
- **显式 GFM (eGFM)**：在解析子空间中构造显式强迫基（例如 $\cos(k_1 x) \phi_q(y)$），驱动多组独立模拟。这些模拟可完全并行。数据拟合时除去强迫贡献 $Ms$，通过最小二乘回归求解约化模型系数。

### 模型形式与拟合
约化模型假设为线性对流-扩散型：
$$
\frac{\partial f}{\partial t} = A f + B \frac{\partial f}{\partial x} + C \frac{\partial^2 f}{\partial x^2},
$$
其中 $A, B, C \in \mathbb{R}^{n \times n}$ 为待定常数（或推广为空间依赖系数）。拟合目标为
$$
\min_{A,B,C} \sum_{i=1}^{N_s} \left\| \frac{\partial f^{(i)}}{\partial t} - M s^{(i)} - A f^{(i)} - B \frac{\partial f^{(i)}}{\partial x} - C \frac{\partial^2 f^{(i)}}{\partial x^2} \right\|_F^2,
$$
可附加正则化与质量守恒约束。

## 6. 论文的主要结论与发现

- eGFM 能够在约化变量和模型形式与底层闭包关系一致时，生成高质量训练数据，并识别准确稳定的约化模型。
- 强迫必须限制在解析子空间内（$n_{\text{ymode}} \le n$），否则会引入不可约化的模态导致模型精度下降。
- 对于均匀剪切流，eGFM 模型的 RMSE 随 $n$ 增加呈谱收敛，性能近似于解析模型和直接拟合模型。

- 速度基变量在高维（$n \ge 3$）时由于出现额外未封闭项，建模效果普遍劣于谱基变量。
