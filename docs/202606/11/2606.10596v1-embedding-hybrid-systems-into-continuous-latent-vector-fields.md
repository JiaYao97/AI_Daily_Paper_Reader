## 摘要
本文证明，当m>2n时，一个n维混合系统可以嵌入到一个m维欧几里得空间中的连续向量场上。这一结果表明，本质上不连续的混合系统通常允许一个连续的外在表示，该表示适用于可微优化。基于这一存在性定理，我们展示了在潜在空间和状态空间都带有一致性损失的潜在神经ODE能够准确恢复混合系统的流。大量实验表明，所提出的方法在仅从时间序列数据学习具有不同几何形状的混合系统方面优于现有方法。

## Abstract
This work proves that an $n$-dimensional hybrid system can be embedded into an $m$-dimensional Euclidean space equipped with a continuous vector field on its embedded image whenever $m>2n$. This result suggests that an intrinsically discontinuous hybrid system generically admits a continuous extrinsic representation that is well-posed for differentiable optimization. Building on this existence theorem, we show that a latent Neural ODE with consistency loss in both the latent and state space can accurately recover the flow of hybrid systems. Extensive experiments suggest the proposed method outperforms the existing method in learning hybrid systems with varying geometries from only time series data.

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：混合系统（Hybrid Automata）通过连续时间向量场与离散状态重置相结合，能够建模大量物理与信息物理过程（如腿式机器人、任务与运动规划）。
- **核心问题**：混合系统在状态重置点存在固有非光滑或间断性，导致其状态演化不适用于可微优化（例如通过神经网络学习）。传统方法要么需要复杂组合的模态选择，要么在事件函数初始化差时出现病态。
- **研究动机**：探索是否可以为混合系统构造一个**连续的外在表示**，使得原本不连续的内在动力学能被连续向量场描述，从而为可微优化提供良好的基础。
- **整体含义**：本文证明了当嵌入空间维度 $m>2n$ 时，$n$ 维混合系统可以嵌入到一个具有连续向量场的 $m$ 维欧几里得空间中。这一理论保证为后续基于神经ODE的学习框架提供了存在性基础，使得从时间序列数据学习混合系统成为可能且更鲁棒。

## 2. 方法论：核心思想、关键技术细节、算法流程

- **核心思想**：利用微分几何中的**横截性理论（Transversality Theory）** 和**预像定理**，证明存在一个连续映射 $f: M \to \mathbb{R}^m$ 同时满足三个条件：
  - (C-1) 边界值恒等：$f(x) = f(r(x)), \forall x \in S$（$S$ 为守卫曲面）
  - (C-2) 速度相容性：$Df(x)V(x) = Df(r(x))V(r(x)), \forall x \in S$
  - (C-3) $f$ 在商空间 $M_H = M/{\sim}$ 上诱导一个嵌入
- **关键技术细节**：
  - 首先在守卫曲面 $S$ 上通过 Whitney 嵌入定理选取嵌入 $g:S\to \mathbb{R}^m$，确定边界值。
  - 利用**衣领坐标（collar coordinates）** 定义一阶泰勒扩展 $\bar{f}_S(x,t)=f_S(x)+t g_S(x)$，类似地在 $R=r(S)$ 侧做扩展。
  - 通过条件 (C-2) 由 $g_S$ 唯一确定 $g_R$，确保导数连续。
  - 应用 Parametric Transversality Theorem 证明：当 $m>2n$ 时，存在一个**残差集**（generic set）的 $g_S$ 使得局部构造是浸入且单射的。
  - 通过相对扰动将局部性质扩展到整个流形 $M$，保证全局嵌入。
- **算法流程（CHyLL++）**：
  - 使用编码器 $f_\theta: M \to \mathbb{R}^m$ 将状态映射到潜在空间。
  - 使用潜在向量场 $V_\theta: \text{Img}(f_\theta) \to \mathbb{R}^m$ 通过神经ODE积分得到潜在轨迹 $\hat{z}_k$。
  - 使用解码器 $f_\theta^{-1}: \text{Img}(f_\theta) \to M$ 重构状态。
  - 损失函数包含多个项：
    - 状态空间一致性损失 $L_x = \text{MSE}(f_\theta^{-1}(\hat{z}_k), x_k)$
    - 潜在空间一致性损失 $L_z = \text{MSE}(\hat{z}_k, f_\theta(x_k))$
    - 粘合损失 $L_g$（强制状态重置点潜在表示相等）
    - 速度相容性损失 $L_v$（强制重置点速度相等）
    - 协方差正则化 $L_c$（防止潜在空间坍缩）
  - 采用**滚动课程学习**：从短片段逐步延长到长片段。
