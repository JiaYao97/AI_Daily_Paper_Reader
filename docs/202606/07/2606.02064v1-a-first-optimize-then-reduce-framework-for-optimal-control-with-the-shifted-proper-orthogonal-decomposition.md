## 摘要
克服科尔莫戈罗夫障碍，为传输主导问题构建高效的降阶模型仍是一项挑战。这阻碍了将它们作为偏微分方程的计算廉价代理模型用于最优控制问题。由于此类问题需要多次计算完整的偏微分方程，使用降阶代理模型可以加快整个最优控制问题的求解。受此启发，本文探索了在最优控制背景下使用非线性模型降阶技术——移位本征正交分解。为此，我们研究了先优化后降阶框架，即先构造完整偏微分问题的最优性系统，然后用降阶模型逼近该最优性系统。我们针对受一维线性平流方程约束的线性二次型最优控制问题考虑该框架，并将移位本征正交分解方法的计算性能与标准本征正交分解进行了比较。

## Abstract
Overcoming the Kolmogorov barrier for constructing efficient reduced-order models (ROMs) for transport-dominated problems remains a challenge. This has impeded their use as computationally cheap surrogates for partial differential equations (PDEs) associated with optimal control problems. Since such problems require multiple computations of the full PDE, employing their reduced-order surrogates instead could speed up the overall optimal control problem. Motivated by this idea, in this paper we explore the use of a nonlinear model-order reduction technique, namely, the shifted proper orthogonal decomposition (sPOD) in an optimal control context. In doing so, we explore the framework of first-optimize-then-reduce (FOTR) where an optimality system for the full PDE problem is constructed first, followed by approximating the optimality system with reduced-order models. We consider this framework for a linear quadratic optimal control problem constrained by a 1D linear advection equation and compare the computational performance of the sPOD method against the use of the standard POD.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 2.1 框架选择：FOTR（先优化后降阶）

- **基本思路**：先对全阶 PDE 约束的最优控制问题应用拉格朗日方法，导出一阶最优性条件（OC FOM），即状态方程、伴随方程和最优性条件构成的耦合系统；再对 OC FOM 中的状态和伴随变量分别进行模型降阶近似。
- **与 FRTO 对比**：与先前工作的“先降阶后优化”（FRTO）框架不同，FOTR 在模型约简前已经获得了全阶最优性条件，理论上可保持最优性的结构（但对 sPOD 而言 FOTR 与 FRTO 并不等价）。

### 2.2 POD‑Galerkin 近似（作为基准方法）

- **状态近似**：用正交基函数 $\{\phi_i\}$ 和时间系数 $\alpha_i(t)$ 线性展开状态 $q(t)$，通过最小化误差构造 POD 基；再通过 Galerkin 投影得到线性常微分方程状态 ROM。
- **伴随近似**：类似地，对伴随变量 $\lambda(t)$ 使用另一组正交基 $\{\psi_i\}$ 展开，得到伴随 ROM。
- **控制最优性条件**：$\mu u(t) + B_{\ell_a}^\top \lambda_{\ell_a}(t) = 0$。
- **特点**：最终得到完全线性的降阶最优性系统（OC POD‑G）。

### 2.3 sPOD‑Galerkin 近似（核心创新）

- **非线性 ansatz**：状态 $q(t)$ 近似为 $\sum_{i=1}^{\tilde\ell} \alpha_i(t) T(z(t)) \phi_i$，其中 $T(z)$ 是用于追踪波传播的周期性移位算子，$z(t)$ 为时变移位，$\phi_i$ 为定常模式。伴随变量采用类似的移位 ansatz，但允许使用不同移位 $z_{\ell_a}(t)$。
- **ROM 推导**：
  - 对状态方程：将 ansatz 代入全阶状态方程，通过 Galerkin 投影得到一组非线性常微分方程（含 $\dot\alpha$ 和 $\dot z$），状态矩阵依赖移位 $z(t)$ 和状态变量 $\alpha$。
  - 对伴随方程：同样推导出伴随 ROM，其中耦合项包含移位差 $z_{\ell_a}-z$ 和时变目标状态 $q_d$。
  - 控制最优性条件：$\mu u(t) + \tilde B(z_{\ell_a}(t))^\top \lambda_{\ell_a}(t) = 0$。
- **关键技术细节**：
  - 移位算子 $T(z)$ 为周期性的平移算子，其导数满足 $T'(z)\phi = -T(z)\phi'$。
  - 为保证伴随终点条件 $\lambda(T)=0$，令 $\lambda_{\ell_a}(T)=0$，而 $z_{\ell_a}(T)$ 可任意选取。
  - 由于 ROM 空间低维但部分项（如 $\tilde q_d$）仍需依赖全阶网格，作者采用预计算/插值策略处理 $B_1(z)$、$B_2(z)$、$\tilde B(z_{\ell_a})$，以降低在线计算成本。
  - 伴随方程系数矩阵在终点处奇异，通过添加小量 $\epsilon=10^{-14}$ 正则化。

### 2.4 算法流程（Algorithm 1）

1. 初始化控制 $u_0$。
2. 迭代优化过程中，每隔若干步或步长选择失败时，**细化**基底：求解全阶状态和伴随快照，构造 sPOD 基（状态和伴随分别构造）。
3. 使用当前 ROM 按时间积分求解降阶状态方程和降阶伴随方程。
4. 计算梯度，通过双回溯法或 Barzilai‑Borwein 方法确定步长，更新控制。
5. 满足终止条件（达到最大迭代次数或梯度下降比例小于阈值）后输出最终控制。

1. **模式效率**：sPOD‑G 达到 FOM 成本所需的模式数比 POD‑G 少一个数量级（单倾斜：25 vs 300；双倾斜：45 vs 300）。

3. **计算时间**：当前实现下，FOM 最快，sPOD‑G 通常比 POD‑G 更慢（尤其伴随求解和梯度计算），这主要是因为 sPOD‑G 的 ROM 方程非线性且部分项仍依赖全阶网格，同时基底构造和移位采样引入了额外开销。

5. **可行性**：首次展示了 sPOD 在 FOTR 框架下可用于求解最优控制问题，尽管存在计算瓶颈，但为后续改进提供了方向。

## 8. 不足与局限

- **问题局限**：仅测试了一维线性平流方程，对高维、非线性、多物理场或存在激波的问题未作验证，sPOD 的泛化性未知。
- **伴随移位假设**：假设伴随移位与状态移位相同（Remark 3.2），该假设仅适用于控制不显著改变传播速度的简单情形，具有局限性；若需独立计算伴随移位会大幅增加开销。
- **计算效率有待改进**：sPOD‑G 的 ROM 求解时间反而比 FOM 长，且伴随求解和梯度计算占总时长比例过大，这与降阶模型“加速”的初衷相悖。作者建议使用超减法（DEIM）等加速策略，但本文未实施。
- **模式选择策略单一**：模式数扫描中状态与伴随模式数保持相等，未能展示各自独立选择的效果；容差实验虽然独立选择，但未与 POD 进行对比。
- **数值噪声风险**：伴随方程系数矩阵正则化（加 $\epsilon=10^{-14}$）可能引入微小误差，但作者未分析其对优化结果的影响。
- **未讨论鲁棒性**：未测试不同初始控制、不同正则化参数 $\mu$、不同网格分辨率下的表现。
