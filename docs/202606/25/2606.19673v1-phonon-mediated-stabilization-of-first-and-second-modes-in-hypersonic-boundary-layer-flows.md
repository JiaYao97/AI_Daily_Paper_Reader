## 摘要
层流向湍流转捩延迟是高超声速边界层流动中的一个关键挑战。不稳定的扰动——最显著的是第一和第二模态——触发了湍流的产生，并对高超声速运输构成了基本的技术障碍。虽然现有的控制策略针对第二模态，但长期以来同时抑制第一模态似乎在物理上是不可能的。本文介绍了一种新的流动控制概念，其中通过地下声子工程定制壁面压力与速度波动之间的相位关系，以同时控制两种模态。结果是显著降低阻力并减轻与湍流相关的极端热载荷。

## Abstract
Laminar-to-turbulent transition delay is a key challenge in hypersonic boundary-layer flows. Unstable disturbances-most prominently the first and second modes-trigger the onset of turbulence and pose a fundamental technological barrier to hypersonic transport. While existing control strategies target the second mode, simultaneous mitigation of the first mode has long appeared physically impossible. A new flow-control concept is introduced in which phase relations between wall pressure and velocity fluctuations are tailored using subsurface phonon engineering to control both modes concurrently. The outcome is substantial drag reduction and alleviation of the extreme thermal loads associated with turbulence.

## 1. 核心问题与整体含义（研究动机与背景）

- **核心问题**：高超声速边界层中层流向湍流的转捩会急剧增加表面传热和摩擦阻力，是制约高超声速飞行器性能、热防护系统及经济可行性的关键技术瓶颈。在 Ma ≈ 4–6 的巡航范围内，转捩主要由 Mack 第一模态（斜波，非零展向波数 β ≠ 0）和第二模态（二维波，β = 0）的不稳定性驱动。
- **已有策略的局限**：现有被动控制手段（如多孔壁面、声学超表面）仅能有效稳定第二模态，但往往导致第一模态失稳；多位知名研究者曾推测，在不改变平均流（如壁面冷却）的前提下，通过被动壁面处理同时稳定第一和第二模态是不可能的。
- **本文突破**：首次提出利用亚表面声子工程（phonon engineering）对壁面压力涨落与壁面法向速度涨落之间的**相位关系**进行局部调控，实现对两种模态的**同时稳定**。该工作为高超声速层流控制提供了全新范式，有望显著降低阻力与热载荷，提升飞行器经济性。

## 2. 方法论：核心思想、关键技术细节、算法流程

### 核心思想
- 通过亚表面声子结构（PSub）实现预定的频率响应函数 $H(\omega)$，建立壁面压力扰动 $\tilde{p}_w$ 与壁面法向速度扰动 $\tilde{v}_w$ 之间的复数关系：
  $$\tilde{v}_w(x,\omega) = H(\omega) \, \tilde{p}_w(x,\omega)$$
- 关键控制参量是相位差 $\Delta\phi_{p',v'} = \measuredangle(\tilde{p}_w, \tilde{v}_w)$。通过设计 $H(\omega)$ 的相位特性，可以**同时抑制第一和第二模态**的扰动增长。
- 响应量级：壁面压力扰动约 10–100 Pa，壁面声子位移约 0.1 μm，与 MEMS 平台兼容。

### 关键技术细节
- **模型假设**：PSub 仅允许法向运动（类似于“钢琴键”），面内位移 $ \tilde{u}_w = \tilde{w}_w = 0$。
- **幅度标度**：最优幅度约为 $|H(\omega)| \sim 1/(\rho_e V_e)$，其中 $\rho_e$、$V_e$ 为边界层边缘密度和速度。
- **相位范围**：通过单输入单输出（SISO）系统可实现 $\Delta\phi \in [0^\circ, 90^\circ] \cup [270^\circ, 360^\circ]$；更宽范围（$90^\circ$–$270^\circ$）需多输入多输出（MIMO）系统。
- **实现方式**：利用阻尼弹簧-质量-阻尼器系统模拟 PSub 频率响应，自然频率设在 40 kHz 和 120 kHz，阻尼比分别为 0.13 和 0.09；宽带响应可通过优化设计近似实现。

### 算法流程（文字描述）
1. **局部稳定性分析（LST）**：在 Mach 5.35 绝热平板边界层中，提取下游某站位（$x=0.6$ m）的平均流剖面，采用空间行进波假设 $q'(x,y,z,t) = \tilde{q}(y) \exp[i(- \alpha x + \omega t + \beta z)]$ 计算空间放大率 $-\alpha_i$。
2. **相位扫描**：固定 $|H|$，改变 $\Delta\phi$，找出对第一、第二模态最有利的相位区间（第一模态最优 $\approx 40^\circ–90^\circ$，第二模态最优 $\approx 90^\circ$ 或 $270^\circ$）。
3. **单频驱动 DNS**：在壁面施加局部弹簧-质量-阻尼器边界条件，模拟 50 kHz（第一模态）和 100 kHz（第二模态）扰动，观察壁面压力信号衰减。
4. **宽带优化**：针对频域 $[10,200]$ kHz 和波角 $[0^\circ,170^\circ]$ 设计 $H(\omega)$，使其相位在 $f_1=10$ kHz 到 $f_2=200$ kHz 间线性变化（$0^\circ \to 170^\circ$），幅度近似恒定。

## 6. 主要结论与发现

1. **首次实现被动同时稳定第一和第二模态**：通过局部壁面相位控制（PSub），在 Mach 5.35 边界层中成功抑制了两种模态的扰动增长。
2. **稳定机理**：Chu 能量预算分析表明，主要机制是**减少了从平均流向扰动场的能量传递（抑制了 Reynolds 应力生产和熵生产项）**，而非单纯增加耗散。壁面法向通量项 $F_{\text{tot}}$ 并非主导因素。
3. **宽带有效**：在贴近真实飞行条件的宽带扰动环境下，PSub 能使已转捩的流动重新层流化（$c_f$ 恢复至层流值），证明了该概念的实用性。
4. **相位控制能力**：通过 MIMO 系统可实现 $\Delta\phi \in [90^\circ, 270^\circ]$ 的相位，从而独立优化对第一和第二模态的响应。
