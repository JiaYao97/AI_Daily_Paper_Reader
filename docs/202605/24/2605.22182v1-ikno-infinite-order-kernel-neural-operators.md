## 摘要
神经算子因其灵活性和强大的泛化能力，在现代科学计算中取得了显著成功。然而，现有模型主要依赖一阶核积分近似，严重限制了其表达能力。为解决这一问题，我们提出了无限阶核神经算子（IKNO），通过无限阶核积分构建神经算子，并给出了优雅的封闭形式有限近似。我们发展了两种互补的无限阶神经算子构造：IKNO-Vanilla，通过Kronecker特征分解在产品网格上应用全核预解式；以及IKNO-TP，一种替代的张量积算子，组合各轴预解式。此外，我们为IKNO的两种变体开发了快速计算方案，在保持高计算效率的同时实现了出色的全局信息聚合。在经验上，我们在具有任意输入形状的时变和时不变基准上（包括大规模工业数据集）评估了我们的IKNO。大量实验表明，IKNO方法在几乎所有基准数据集上持续达到了最先进的精度，并实现了显著改进，同时保持了对极大点云的可扩展性。

## Abstract
Neural operators have achieved significant success in modern scientific computing due to their flexibility and strong generalization capabilities. Existing models, however, primarily rely on first-order kernel integral approximations, which severely limit their expressivity. To address this, we propose the Infinite-order Kernel Neural Operator (IKNO), which constructs neural operators via infinite-order kernel integrals and admits an elegant closed-form finite approximation. We develop two complementary infinite-order neural operator constructions: IKNO-Vanilla, which applies the full-kernel resolvent on the product grid via Kronecker eigendecomposition, and IKNO-TP, an alternative tensor-product operator that composes per-axis resolvents. Furthermore, we develop fast computation schemes for both variants of IKNO, which achieve outstanding global information aggregation while maintaining high computational efficiency. Empirically, we evaluate our IKNO on both time-dependent and time-independent benchmarks with arbitrary input shapes, including large-scale industrial datasets. Extensive experiments demonstrate that the IKNO method consistently achieves the SOTA accuracy with significant improvements on nearly all benchmark datasets while maintaining scalability to very large point clouds.

# 论文总结：IKNO: Infinite-order Kernel Neural Operators

## 1. 核心问题与研究动机

现有神经算子（如FNO、GINO、GAOT等）主要采用**一阶核积分近似**，即只对输入函数进行一次全局核聚合。论文通过初步实验（Poisson-C-Sines基准）发现，将核传播阶数从 $p=1$ 增至 $p=4$ 时，误差单调降低（从2.49%降至2.13%），表明**高阶核传播能更有效聚合全局信息**。由此提出核心问题：能否构造**无限阶核积分算子**，实现所有传播阶数的完备聚合，从而突破一阶近似的表达力瓶颈？

## 2. 方法论：无限阶核神经网络算子（IKNO）

### 核心思想
- 定义 $n$ 阶核积分算子 $I_n[v](x) = \int_X \cdots \int_X k(x, y_1) \cdots k(y_{n-1}, y_n) v(y_n) \, dy_1 \cdots dy_n$。
- 无限阶核积分算子为级数 $I_\infty[v] = I_1 + \alpha I_2 + \alpha^2 I_3 + \cdots$。
- 在离散采样下，该级数可写成矩阵形式 $\hat{I}_\infty[v](x) = k(x)^\top (I + \alpha K + \alpha^2 K^2 + \cdots) V(Y)$。
- 当 $\rho(\alpha K) < 1$ 时，级数收敛为闭式解：$\hat{I}_\infty[v](x) = k(x)^\top (I - \alpha K)^{-1} V(Y)$，即**核预解式**。

### 两种算子构造
- **IKNO-Vanilla**：直接在 $d$ 维产品网格的全核矩阵 $K = K_1 \otimes \cdots \otimes K_d$ 上应用预解式 $(I_M - \alpha K)^{-1}$，保留所有跨维耦合项。
- **IKNO-TP**：采用逐轴张量积方式，定义 $K^{\text{TP}}_\infty = (I_N - \alpha K_1)^{-1} \otimes \cdots \otimes (I_N - \alpha K_d)^{-1}$，对各轴独立求解预解式后再张量组合，隐式引入**函数正则化**。

### 快速计算方案
- 利用Kronecker结构：对每个 $K_j$ 做特征分解 $K_j = U_j \Lambda_j U_j^\top$，则将 $M \times M$ 矩阵求逆（复杂度 $O(N^{3d})$）降为 $O(d \cdot N^3)$ 预处理 + $O(d \cdot N \cdot M)$ 前向传播。
- IKNO-Vanilla通过全分解公式 $(I - \alpha \otimes_j K_j)^{-1} = ( \otimes_j U_j ) \operatorname{diag}(1/(1 - \alpha \prod_j \lambda_{i_j}^{(j)})) (\otimes_j U_j)^\top$ 实现。
- IKNO-TP直接对每个 $I_N - \alpha K_j$ 求逆（$d$ 个 $N \times N$ 矩阵），再通过逐轴张量乘实现。

### 可学习多尺度基核
- 对每个维度 $j$ 设计可学习核：$k_j(x_j, y_j) = c_j \left[ \exp(-|\beta_j (x_j - y_j)|^2) + \exp(-|\gamma_j (x_j - y_j)|) \right]$，其中 $c_j, \beta_j, \gamma_j$ 可学习，从而自适应调整各向异性尺度。

### 架构
- **编码器**：带几何增强（NeRF式位置编码）的MLP将输入点投影到潜在网格，并经无限阶核算子 $K_\infty$ 聚合。
- **处理器**：Transformer编码器（12层，8头，$d_{\text{model}}=256$）进行全局混洗。
- **解码器**：再经无限阶核算子映射回任意查询点，最后通过MLP输出物理量。
