## 摘要
基于经典本征正交分解（POD）的混沌流伽辽金投影模型通常需要大量模态以及稳定化或封闭项才能达到足够的精度和长期稳定性。我们提出了一种新颖的可微编程框架，该框架在不增加模态数量或引入额外封闭项的情况下稳定低秩POD-伽辽金模型，从而同时实现高效率和精度。模型稳定化是通过使用可微编程调整POD-伽辽金中的线性和二次张量来实现的，并在短期轨迹数据上进行训练。本研究的一个关键发现是，对于混沌系统，纯逐点轨迹损失函数会导致较差的长期精度。相比之下，结合轨迹误差和基于物理的能量守恒项的混合损失函数能提供优越的长期性能。我们在雷诺数30,000的混沌顶盖驱动方腔流上演示了该方法。与经典POD-伽辽金方法相比，稳定化ROM在计算成本上实现了数量级降低：仅需20个模态就能保持准确和稳定，而经典ROM则需要80个POD模态。

## Abstract
Classical Proper Orthogonal Decomposition (POD)-based Galerkin projection models of chaotic flows typically require a large number of modes as well as stabilization or closure terms to achieve adequate accuracy and long-term stability. We present a novel differentiable programming framework that stabilizes low-rank POD-Galerkin models without increasing the number of modes or introducing additional closure terms, thereby delivering both high efficiency and high accuracy. Model stabilization is achieved by tuning the linear and quadratic tensors in the POD-Galerkin using differentiable programming, trained on short-term trajectory data. A key finding of this study is that a purely point-wise trajectory-based loss function yields poor long-term accuracy for chaotic systems. In contrast, a hybrid loss function that combines trajectory error with a physics-based conservation-of-energy term provides superior long-term performance. We demonstrate the approach on a chaotic lid-driven cavity flow at Re = 30,000. The stabilized ROM achieves an order-of-magnitude reduction in computational cost compared with the classical POD-Galerkin method: it remains accurate and stable with only 20 modes, whereas the classical ROM requires 80 POD modes.

- **核心问题**：经典 POD-Galerkin 投影降阶模型（ROM）在处理混沌/湍流流动时，由于截断丢失了小尺度耗散模态，导致长期积分能量过度积累、数值发散。传统方案需要保留大量模态（如 80 个）或引入额外稳定化/封闭项，但前者增加计算成本（成本随模态数 $N$ 呈 $O(N^3)$ 增长），后者往往只能改善短期精度而无法保证长期稳定性。

### 2. 论文提出的方法论

- **核心思想**：保留 POD-Galerkin 投影导出的 ODE 结构中的二次项（由投影内积确定且不包含可调参数），将线性和常数系数 $C_i$、$L_{ij}$ 设为可训练参数 $p$，通过微分编程（Differentiable Programming）进行端到端优化。
- **关键技术细节**：
  - **ROM ODE 形式**：
    $$ \dot{a}_i = C_i^p + \sum_{j=1}^N L_{ij}^p a_j + \sum_{j=1}^N \sum_{k=1}^N Q_{ijk} a_j a_k $$
    其中 $Q_{ijk}$ 从经典 Galerkin 投影获得并冻结；$C_i^p$、$L_{ij}^p$ 为可学习的张量。
  - **初始化**：将 $C_i^p$、$L_{ij}^p$ 初始化为经典 Galerkin 投影的原始系数，使训练从物理合理起点开始。
  - **损失函数**：提出两种形式
    - 纯轨迹损失 $\mathcal{L}_{\text{traj}}$：MSE 归一化的预测系数与 DNS 系数的偏差。
    - 混合损失 $\mathcal{L}_{\text{total}} = \lambda \mathcal{L}_{\text{traj}} + (1-\lambda)\mathcal{L}_{\text{energy}}$，其中 $\mathcal{L}_{\text{energy}}$ 为预测与 DNS 之间时间平均湍动能相对差的平方。
  - **优化方法**：使用反向模式自动微分（伴随灵敏度法）计算梯度，采用 Adam 优化器（初始学习率 $1\times10^{-3}$，必要时降至 $1\times10^{-4}$），并在训练过程中利用验证集实施早停，避免过拟合。
  - **数据预处理**：将 $C_i^p$、$L_{ij}^p$ 除以各自最大绝对值进行归一化；轨迹损失除以全局 RMS 因子 $\sigma$；能量损失除以参考平均湍动能。
- **算法流程**：
  1. 从 DNS 快照提取 POD 基础，计算经典 Galerkin 系数 $C_i$、$L_{ij}$、$Q_{ijk}$。
  2. 初始化可训练参数 $p$ 为 $C_i$、$L_{ij}$。
  3. 前向传播：用 ODE 积分器（四阶 Runge-Kutta）在训练时间窗（25 个无量纲时间单位）内计算轨迹。
  4. 计算损失（纯轨迹或混合），反向传播更新 $p$。
  5. 验证集监视泛化，满足早停条件则恢复最优参数。
  6. 最终模型用于长期预测。

### 6. 论文的主要结论与发现

1. **稳定效果**：DPG-ROM 框架成功稳定了高度截断的 20/30 模态 ROM，而经典 GP-ROM 至少需要 80 模态才能产生有界预测。
2. **纯轨迹损失局限**：仅用 $\mathcal{L}_{\text{traj}}$ 训练的模型在短期紧密跟踪 DNS，但长期仍出现小幅能量漂移（尤其 20 模态），说明逐点损失无法强制模型学会正确的耗散机制。
3. **混合损失优势**：引入能量约束（$\mathcal{L}_{\text{energy}}$）后，20 模态的 DPG-ROM 在长期保持与 DNS 一致的湍动能水平，相空间紧致，频域能量分布略有改变但整体稳定。最优混合系数 $\lambda=0.5$。
4. **校准方法失效**：传统线性校准因只优化瞬时残差，对混沌系统长期稳定性改善有限，甚至劣于未校准的 GP-ROM。
5. **计算效益**：20 模态 DPG-ROM 在线预测比 DNS 快约 3200 倍，比传统 80 模态 ROM 也大幅加速（成本随模态数立方增长）。

### 7. 优点

- **方法论创新**：在保持物理驱动结构（二次项冻结）的同时，利用可微编程数据驱动调整线性和常数项，是“物理信息+数据驱动”的有机融合，避免了纯黑箱模型的不解释性。

- **损失函数设计**：混合损失以较小代价引入全局能量约束，显著提升了长期稳定性，且发现 $\lambda$ 存在最优值（0.5），提供了实际调参指导。
- **诊断分析完善**：通过湍动能平衡（$\bar{P}$、$\bar{D}$）、特征值移位、PSD 等多角度解释了稳定化机制，增加了方法可信度。

### 8. 不足与局限

- **流动问题单一**：仅验证了二维 LDC 单一算例。该方法在其他复杂流动（三维、非定常、分离流、高雷诺数）上的泛化能力未证明。
