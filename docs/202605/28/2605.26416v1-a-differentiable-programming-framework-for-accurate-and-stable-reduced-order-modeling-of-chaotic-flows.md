## 摘要
基于经典本征正交分解（POD）的Galerkin投影模型在混沌流中通常需要大量模态以及稳定化或闭合项才能达到足够的精度和长期稳定性。我们提出了一种新颖的可微编程框架，它在不增加模态数量或引入额外闭合项的情况下稳定低秩POD-Galerkin模型，从而实现高效且高精度。模型稳定化是通过使用可微编程调整POD-Galerkin中的线性和二次张量来实现的，并基于短期轨迹数据进行训练。本研究的一个关键发现是，纯粹的逐点轨迹损失函数会导致混沌系统长期精度较差。相比之下，结合轨迹误差与基于物理的能量守恒项的混合损失函数能够提供优越的长期性能。我们在雷诺数为30,000的混沌顶盖驱动空腔流上展示了该方法。与经典POD-Galerkin方法相比，稳定化ROM的计算成本降低了一个数量级：仅用20个模态即可保持精确和稳定，而经典ROM需要80个POD模态。

## Abstract
Classical Proper Orthogonal Decomposition (POD)-based Galerkin projection models of chaotic flows typically require a large number of modes as well as stabilization or closure terms to achieve adequate accuracy and long-term stability. We present a novel differentiable programming framework that stabilizes low-rank POD-Galerkin models without increasing the number of modes or introducing additional closure terms, thereby delivering both high efficiency and high accuracy. Model stabilization is achieved by tuning the linear and quadratic tensors in the POD-Galerkin using differentiable programming, trained on short-term trajectory data. A key finding of this study is that a purely point-wise trajectory-based loss function yields poor long-term accuracy for chaotic systems. In contrast, a hybrid loss function that combines trajectory error with a physics-based conservation-of-energy term provides superior long-term performance. We demonstrate the approach on a chaotic lid-driven cavity flow at Re = 30,000. The stabilized ROM achieves an order-of-magnitude reduction in computational cost compared with the classical POD-Galerkin method: it remains accurate and stable with only 20 modes, whereas the classical ROM requires 80 POD modes.

# 论文总结：一种用于混沌流准确且稳定的降阶建模的可微编程框架

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：基于经典本征正交分解（POD）的 Galerkin 投影降阶模型（GP-ROM）在混沌/湍流流动中通常需要保留大量模态（如 80 个）才能实现长期稳定，且常需额外闭合项或稳定化方法。这导致计算成本随模态数 $N$ 呈 $O(N^3)$ 增长，限制了实际应用。
- **背景**：混沌流的 ROM 存在“模态截断 → 小尺度耗散丢失 → 动能过预测 → 长期发散”的根本困难。已有稳定化方法（如最小二乘、Tikhonov 正则化、校准法等）要么依赖瞬时线性化残差，忽略累积轨迹误差，要么需要额外项或修改方程结构。
- **整体含义**：本文提出一种**可微编程 Galerkin 投影 ROM（DPG-ROM）**，在不增加模态数、不引入闭包项的前提下，通过可微编程直接调谐 GP-ROM 的线性和常数系数，从而稳定低秩模型。核心发现是**纯轨迹损失对混沌系统长期预测不足**，而**引入能量守恒约束的混合损失**可显著提升长期精度与稳定性。在雷诺数 $Re=30{,}000$ 的顶盖驱动空腔流（LDC）上，仅用 20 个模态即达到经典方法 80 个模态的精度，计算成本降低一个数量级。

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想
- 保持 GP-ROM 的二次非线性项（由 Galerkin 投影得到）不变，将线性系数 $L_{ij}$ 和常数系数 $C_i$ 视为**可训练参数**，通过可微编程（微分编程）优化，使其“吸收”截断模态的耗散效应。
- 训练目标：在短时间窗口（25 个非维时间单位）内最小化预测系数轨迹与 DNS 参考轨迹的误差。关键改进：引入混合损失函数，包含轨迹误差项和时均湍动能（TKE）误差项。

### 2.2 关键技术细节
1. **基线 GP-ROM ODE 形式**：
   $$\dot{a}_i = C_i + \sum_{j=1}^N L_{ij} a_j + \sum_{j=1}^N\sum_{k=1}^N Q_{ijk} a_j a_k$$
   其中 $C_i$、$L_{ij}$ 由内积计算得到，$Q_{ijk}$ 由 Galerkin 投影固定。

2. **DPG-ROM 参数化**：
   - 将 $C_i$ 和 $L_{ij}$ 替换为可训练参数 $C_i^p$ 和 $L_{ij}^p$，初始化时等于投影值。
   - 模型通过求解 ODE 得到预测系数 $a_i^{\text{pred}}(t)$，与参考值比较计算损失。

3. **损失函数**：
   - **轨迹损失** $L_{\text{traj}}$：归一化的逐点均方误差：
     $$L_{\text{traj}} = \frac{1}{N T}\sum_{n=1}^T\sum_{i=1}^N\left(\frac{a_{i,n}^{\text{pred}} - a_{i,n}^{\text{ref}}}{\sigma}\right)^2$$
     其中 $\sigma$ 为全局 RMS 缩放因子。
   - **能量损失** $L_{\text{energy}}$：比较预测和参考的时均湍动能 $\bar{K}$：
     $$L_{\text{energy}} = \left(\frac{\bar{K}^{\text{pred}} - \bar{K}^{\text{ref}}}{\bar{K}^{\text{ref}}}\right)^2$$
   - **混合损失**：
     $$L_{\text{total}} = \lambda L_{\text{traj}} + (1-\lambda) L_{\text{energy}},\quad \lambda\in[0,1]$$
     最优 $\lambda=0.5$ 由 NRMSE 扫描确定。

4. **训练流程**：
   - 使用 Adam 优化器（初始学习率 $10^{-3}$，可降至 $10^{-4}$）。
   - 训练窗口：2500 个快照（25 时间单位），60% 用于训练，40% 用于验证（通过非重叠片段，带指数加权采样）。
   - 采用早停（early stopping）防止过拟合，恢复至验证损失最低的模型参数。
   - 系数归一化：$C_i$ 和 $L_{ij}$ 分别除以最大绝对值，使初始参数在 $[-1,1]$ 内。
   - 反向传播通过 ODE 积分器（使用伴随灵敏度方法和自动微分）。

5. **能量组分分析**：利用时均湍动能率方程 $\partial \bar{K}/\partial t = \bar{P} + \bar{D} + \bar{C} + \bar{T} \approx 0$，重点考察生产项 $\bar{P}$ 和耗散项 $\bar{D}$ 的平衡，以判断模型是否学会物理的耗散机制。

## 6. 主要结论与发现

1. **DPG-ROM 可成功稳定高度截断的 GP-ROM**：仅 20 模态即能产生长期有界预测，而经典 GP-ROM 需要至少 80 模态。这证明了可微编程作为稳定化机制的有效性。
2. **纯轨迹损失虽优于未校准 GP-ROM，但仍残余长期不稳定性**：由于仅惩罚逐点误差，模型可能在小扰动下收敛到拟合训练窗口但在窗口外发散的参数。20 模态时 DPG-ROM t 仍出现动能尖峰。
3. **混合损失（轨迹+能量）显著提升长期稳定性**：最优 $\lambda=0.5$ 使模型学会物理耗散机制，在 20 模态下实现与 DNS 一致的湍动能水平、紧致的相空间轨迹和持久的长期预测。
4. **经典校准方法（Cal）效果劣于甚至不如未校准 GP-ROM**：因其线性化瞬时残差不考虑累积轨迹误差，在混沌流中错误放大，导致长期表现退化。
5. **能量组分分析**表明：DPG-ROM e 降低了生产和耗散水平，但仍存在不平衡（$\bar{P}>\bar{D}$），指出改进方向。
6. **功率谱分析**显示：DPG-ROM 模型将能量从低频向高频重新分布，表明其学习了新的耗散机制，但低频功率略低于 DNS。
7. **特征值演化**显示训练后线性矩阵特征值向左复平面移动，印证了稳定性增强。

## 7. 优点

- **方法创新性强**：将可微编程用于 ROM 稳定化，保持物理结构（二次非线性项）不变，仅调整线性/常数项，是“物理知识嵌入数据驱动”的典范。
