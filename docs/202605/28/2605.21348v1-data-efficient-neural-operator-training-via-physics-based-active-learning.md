## 摘要
使用神经算子求解偏微分方程显著降低了计算成本，但仍受限于高训练数据需求。主动学习通过迭代方式有选择地获取信息量最大的样本，为此提供了一个自然的缓解框架。我们提出了基于物理的采集——一种新颖的物理信息主动学习算法，利用偏微分方程残差指导数据选择。通过对一维Burgers方程和二维可压缩Navier-Stokes方程进行数值实验，我们验证了该方法。结果表明，在我们的实验中，基于物理的采集始终优于随机采集，并在数据效率上达到了最先进水平。同时，它具有独特的优势，即向训练过程中注入物理归纳偏置，确保计算成本花费在模型物理理解最薄弱的地方。

## Abstract
Solving partial differential equations with neural operators significantly reduces computational costs but remains bottlenecked by high training data requirements. Active learning offers a natural framework to mitigate this by selectively acquiring the most informative samples in an iterative manner. We introduce physics-based acquisition - a novel physics-informed active learning algorithm that leverages the partial differential equation residual to guide data selection. We validate the method by presenting numerical experiments for the 1D Burgers equation and the 2D compressible Navier-Stokes equations. We show that, in our experiments, physics-based acquisition consistently outperforms random acquisition and matches the state of the art in data efficiency. At the same time, it has the unique advantage of injecting a physics inductive bias into the training process, ensuring that simulation cost is spent where the model's physical understanding is weakest.

# 基于物理主动学习的数据高效神经算子训练：论文总结

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心问题**：神经算子（如 Fourier Neural Operator）能够在少量计算代价下求解偏微分方程（PDE），但其训练过程需要大量由高性能数值模拟器生成的训练数据。模拟器本身计算昂贵，导致“鸡生蛋”困境：为了训练代理模型，必须先运行昂贵的模拟器，而代理模型的初衷是替代模拟器。因此，如何以最少的模拟代价（即最少的训练样本）获得高性能代理模型是一个关键挑战。
- **整体含义**：本文旨在通过主动学习（Active Learning）框架，选择最“有价值”的模拟样本加入训练集，从而显著提高神经算子的数据效率。与其他纯数据驱动的主动学习方法不同，本文提出利用 PDE 残差（Physics Residual Error, PRE）作为物理引导的采集函数，使训练过程注入物理归纳偏置，将计算资源导向模型物理理解最薄弱的区域。

## 2. 论文提出的方法论
### 核心思想
- 基于物理残差误差（PRE）衡量代理模型预测解的“不物理”程度。PRE 是 PDE 微分算子作用于近似解后得到的残差 $R$：$R = \mathcal{D}(\hat{u})$。对于精确解，$R \equiv 0$；对于不准确的解，$|R|$ 较大。因此，PRE 可作为模型在特定输入-参数对上的**不确定性**指标 。
- 主动学习过程中，优先采集当前模型预测 PRE 最高的候选样本（初始条件 + PDE 参数），从而直接纠正模型最违反物理的地方。

### 关键技术细节
1. **PRE 的计算**：使用有限差分模板作为卷积核，将 PRE 估计转化为可 GPU 加速的卷积运算，无需访问计算图或模型参数。
2. **采集得分**：对于候选样本 $(\delta, \lambda)$，先利用当前代理模型自回归地展开整个时空轨迹 $\hat{u}(x_i, t)$，然后计算轨迹上所有时空点的绝对 PRE 的平均值：
   $$s(\delta, \lambda) = \frac{1}{N_s N_t} \sum_{i=1}^{N_s} \sum_{t=1}^{N_t} |\text{PRE}(x_i, t)|.$$
3. **批量选择策略**：采用两种策略：
   - **Top-k**：直接选取得分最高的 $k$ 个样本。
   - **Stochastic Batch Active Learning (SBAL)**：对得分施加幂律噪声后随机采样，增加批次多样性。
4. **跨参数残差归一化**：由于不同 PDE 参数下残差尺度不同（如粘性系数变化导致方程缩放改变），需要将候选样本的得分除以**训练集中最近邻参数对应的真实轨迹的 PRE 均值**，以实现跨参数范围的可比性。最近邻由参数空间中的欧氏距离定义。

## 6. 论文的主要结论与发现
- 基于物理的主动学习方法在 Burgers 和 2D Navier-Stokes 上**持续优于随机采样**，且在数据效率上与 LCMD 相当（RMSE 曲线几乎重叠）。

- 实验表明，在给定的参数范围内，PRE 能提供强烈且有用的信号来指导样本选择，但需要适当的跨参数归一化。

## 7. 优点
1. **物理可解释性**：采集准则具有明确的物理意义（模型哪里最不准物理），易于理解和调试。
2. **计算高效**：PRE 可通过卷积快速在 GPU 上计算，无需训练多个模型或进行贝叶斯推断。
3. **即插即用**：作为不确定性度量，可以轻松替换现有的采集函数，且可用于任何可计算 PDE 残差的场景。
4. **改善数据效率**：在实验条件下显著降低所需训练轨迹数量，减少模拟计算成本。
5. **开源性**：基于 AL4PDE 框架实现，有利于复现和扩展。
