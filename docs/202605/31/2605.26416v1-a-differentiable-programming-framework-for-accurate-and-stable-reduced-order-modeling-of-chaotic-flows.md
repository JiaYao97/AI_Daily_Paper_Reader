## 摘要
基于经典本征正交分解（POD）的伽辽金投影模型通常需要大量模态以及稳定化或封闭项才能达到足够的精度和长期稳定性。我们提出了一种新颖的可微编程框架，该框架在不增加模态数量或引入额外封闭项的情况下稳定低秩POD-伽辽金模型，从而同时实现高精度和高效率。模型稳定化是通过使用可微编程调整POD-伽辽金中的线性和二次张量实现的，并在短期轨迹数据上进行训练。本研究的一个关键发现是，纯逐点轨迹损失函数会导致混沌系统长期精度较差。相比之下，结合轨迹误差和基于物理的能量守恒项的混合损失函数提供了更优的长期性能。我们在雷诺数Re=30,000的混沌顶盖驱动方腔流动上演示了该方法。与经典POD-伽辽金方法相比，稳定化ROM的计算成本降低了一个数量级：仅需20个模态即可保持准确和稳定，而经典ROM需要80个POD模态。

## Abstract
Classical Proper Orthogonal Decomposition (POD)-based Galerkin projection models of chaotic flows typically require a large number of modes as well as stabilization or closure terms to achieve adequate accuracy and long-term stability. We present a novel differentiable programming framework that stabilizes low-rank POD-Galerkin models without increasing the number of modes or introducing additional closure terms, thereby delivering both high efficiency and high accuracy. Model stabilization is achieved by tuning the linear and quadratic tensors in the POD-Galerkin using differentiable programming, trained on short-term trajectory data. A key finding of this study is that a purely point-wise trajectory-based loss function yields poor long-term accuracy for chaotic systems. In contrast, a hybrid loss function that combines trajectory error with a physics-based conservation-of-energy term provides superior long-term performance. We demonstrate the approach on a chaotic lid-driven cavity flow at Re = 30,000. The stabilized ROM achieves an order-of-magnitude reduction in computational cost compared with the classical POD-Galerkin method: it remains accurate and stable with only 20 modes, whereas the classical ROM requires 80 POD modes.

## 二、方法论

### 2.1 核心思想
- 保留经典POD-Galerkin ODE中的**非线性二次项**（由POD模式间投影得到），将**线性项系数 $L_{ij}$ 和常数项系数 $C_i$** 设为可训练参数，利用可微编程从DNS数据中学习其修正值，从而补偿因模态截断丢失的小尺度耗散机制。

### 2.2 关键技术细节
1. **模型结构**：
   \[
   \dot{a}_i = C_i^p + \sum_{j=1}^N L_{ij}^p a_j + \sum_{j=1}^N\sum_{k=1}^N Q_{ijk} a_j a_k
   \]
   其中 $C_i^p$、$L_{ij}^p$ 为可学习参数，初始化为经典Galerkin投影值；二次张量 $Q_{ijk}$ 固定不变。

2. **损失函数**：
   - **轨迹损失**：$L_{\text{traj}} = \frac{1}{NT}\sum_{n=1}^T\sum_{i=1}^N \left(\frac{a_{i,n}^{\text{pred}} - a_{i,n}^{\text{ref}}}{\sigma}\right)^2$，逐点MSE。
   - **能量损失**：$L_{\text{energy}} = \left(\frac{\bar{K}_{\text{pred}} - \bar{K}_{\text{ref}}}{\bar{K}_{\text{ref}}}\right)^2$，基于时间平均湍动能 $\bar{K}$。
   - **总损失**：$L_{\text{total}} = \lambda L_{\text{traj}} + (1-\lambda)L_{\text{energy}}$，经超参扫描确定最优 $\lambda=0.5$。

3. **训练流程**：
   - 使用反向模式自动微分（伴随灵敏度法）计算梯度，通过ODE积分器进行端到端训练。
   - 优化器：Adam（学习率 $1\times10^{-3}$ 或 $1\times10^{-4}$）。
   - 训练窗口：2500个快照（25个无量纲时间单位），60%用于训练，40%用于验证。
   - 采用早停策略和系数归一化（将初始参数缩放到[-1,1]）以稳定训练。

## 六、主要结论与发现
1. **DPG-ROM可有效稳定低秩ROM**：20模态DPG-ROM（混合损失）达到了80模态经典GP-ROM的长期稳定性和统计精度，计算成本降低一个数量级。
2. **纯轨迹损失存在局限**：$L_{\text{traj}}$ 仅优化点对点误差，无法强制耗散机制，导致20模态模型在长期仍有小幅能量积累（图5、图6）。验证损失高于训练损失说明模型对混沌轨迹的泛化能力有限。
3. **混合损失显著提升长期稳定性**：引入能量约束后，20模态DPG-ROM e的TKE、相图、频率分布均更接近DNS参考，且避免了纯轨迹模型在 $t\approx100$ 附近的尖峰。
4. **传统校准方法失效**：在高度截断情况下（20模态），校准GP-ROM甚至比未校准GP-ROM表现更差，原因是其线性化残差优化累积了全局误差。
5. **能量分析揭示学习机制**：DPG-ROM e的线性系数特征值向左平移，表明模型学习了更强耗散；但生产-耗散平衡（$\bar{P}\approx\bar{D}$）仍不完美（20模态下 $\bar{P}=0.05$, $\bar{D}=0.01$），提示仍有改进空间。

## 七、优点
- **无需额外修正项**：通过可微编程直接调整ODE系数，保留物理结构（二次非线性项），模型可解释性强。
- **数据高效**：仅需短时间窗口（25时间单位）的DNS数据即可训练稳定长期预测，训练成本低（CPU单核数十分钟）。
- **计算加速显著**：20模态在线推理比DNS快约3200倍，比80模态经典ROM也降低大量计算。
- **混合损失设计合理**：能量项提供物理约束，避免过度拟合混沌轨迹，超参 $\lambda$ 可调提供了灵活平衡。
- **实验诊断全面**：能量平衡、特征值演化、PSD等多角度分析深入揭示了模型的物理一致性。

## 八、不足与局限
- **泛化性不足**：仅测试了一个具体流动（LDC，$Re=30,000$），未在更高雷诺数、三维流、不同几何或非定常边界条件上验证，方法的鲁棒性和适用范围不明确。
- **能量平衡未完全匹配**：20模态DPG-ROM e的生产项 $\bar{P}$ 和耗散项 $\bar{D}$ 仍有偏差（$\bar{P}\approx0.05$, $\bar{D}\approx0.01$），表明能量预算重构尚不完美，可能影响极限环等精细结构。

- **超参敏感**：$\lambda$ 的取值依赖于扫描，不同流动可能需要重新调整，缺乏自动选择方法。
- **计算资源限制**：所有训练在CPU单核上完成，未利用GPU加速，对于更大规模问题（如三维）的扩展性未评估。
